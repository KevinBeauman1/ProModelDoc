Simplified "Just-in-Time" Production Control

Move entities forward in a process only when the next downstream activity is ready to process the entity.

 

The model used for this example, as well as all of the modeling "how-to" examples, can be found in the ProModel Solutions Café (http://www.promodel.com/solutionscafe/howto/).

Solution

  1. Set the input buffer capacity of the downstream activity to zero. (By default, Process Simulator routes entities forward only when there is available capacity.)
Example

A fragile lamp is handed off from a Test operation to a Packaging operation only when the Packager is ready for it. Otherwise the tester holds on to the lamp.

Solution

The input buffer capacity of the Packaging activity is set to 0.
