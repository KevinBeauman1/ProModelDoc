Process Simulator Ribbon

The Process Simulator ribbon is provided with all the options needed to define simulation data, run simulation models, and review model the results. The options include:

  • Attributes Allows you to define attributes that are like tags that contain unique values for entities in your model. For more information, see Attributes.
  • Variables Allows you to create global variables for use in logic or nearly any property field in your model. For more information, see Variables.
  • Arrays Similar to a matrix of global variables that hold data needed in your model. Array data can be initialized from an Excel spreadsheet. For more information, see Arrays.
  • Macros Elements that represent a number or distribution that is repetitively used throughout your model. For more information, see Macros.
  • Subroutines A command that can represent a block of logic or an expression that returns a resulting value, which can be called from most fields and any logic in your model. For more information, see Subroutines.
  • Properties Displays the Properties window for the selected shape in your diagram. For more information, see Shapes.
  • Object Explorer Displays a view of all the objects (shapes and connections) you have defined in your diagram. For more information, see Object Explorer.
  • More Elements Displays a list of additional model elements such as Resource Groups, User Distributions, External Arrivals, and External Files Initialization and Termination Logic that opens their individual windows.
  • Simulate Initiates the simulation of the model. 
  • Simulation Options Opens a dialog for defining simulation run length and other animation properties. For more information see Simulation Options.
  • Simulate Scenarios Runs your defined scenarios.
  • Scenario Manager Opens the Scenario Manager for defining what-if scenarios and their parameters used to experiment and optimize your model. For more information, see Scenario Manager .
  • Output Viewer Launches Output Viewer and loads the results of the active diagram for analysis of your process simulation. For more information, see Output Viewer.
  • Find Opens the Find or drop down Replace window, which allows you to search for (and even replace) text contained in any of Process Simulator's property fields and logic.
  • Calendar Editor Opens the Calendar Editor program, which allows you to define time off and work schedules for resources and activities in your model. For more information, see Calendar Editor.
  • More Tools Displays a list of additional Tools:
    • Create or Install Packages Allows you to create or install Process Simulator package files (.pspkg). Package files contain your Visio document (.vsdx) file and any other supporting files, such as calendar and external arrival files.
    • Import or Export Import or Export simulation data to or and from Excel.
    • Stat::Fit Launches Stat::Fit, which takes the actual historical data of your process, analyzes it, and fits it to statistical distributions that represent your data.
    • Options Opens the Process Simulator Options dialog, which provides some default application settings.
  • Getting Started Opens the Getting Started panel window.
  • Help Gives you access to the Process Simulator online help system as well as the About dialog, which displays product version and licensing information, and access to check for product updates.
