Batching or Consolidating Activities

Two or more entities are batched or consolidated at an activity.

 

The model used for this example, as well as all of the modeling "how-to" examples, can be found in the ProModel Solutions Café (http://www.promodel.com/solutionscafe/howto/). A video demonstrating this solution can be found here.

Solution

  1. Place a buffer on the layout. (Alternatively, you can place an activity on the layout and batch in the input or output buffer of the activity.)
  2. In the Buffer dialog (or Activity Batching dialog), select Batch as the Batching Type.
  3. Enter the quantity to be batched.
Example

In the final Collector inventory buffer of a process sequence, entities are batched into quantities of 20 and then sent to shipping.

Solution

The Batching for the Collector activity is set to Batch with the quantity set to 20. (Notice that entities can be batched by entity type by selecting the By <Entity Type> in the drop down box.)
