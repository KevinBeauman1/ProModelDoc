Defining Logic

Process Simulator logic controls the flow of entities through your process and defines what happens to them along the way. For information and theory on controlling this flow of entities, see Flow Control. In its basic form, logic is defined in the properties dialogs of objects. Defining Basic Logic

All the logic necessary to model a high-level view of a process can be entered through the properties dialogs of the objects in your layout. Right clicking on an object and selecting Properties from the menu will display properties for that object.

 

From the properties dialogs you can define such logic as process times, resource usage, routing decisions, arrival patterns, and other high-level logic. For more information on the logic options available from the properties dialogs, please see Defining a Process.

Defining Advanced Logic

Advanced logic allows you to add more detail to your process by including deeper decision-making, tracking, and control logic.

 

This advanced logic is defined using Free Form Logic or the Logic Builder, which provides an easy interface for entering syntactically correct logic into Process Simulator.
