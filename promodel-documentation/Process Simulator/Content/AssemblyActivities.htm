Assembly Activities

One or more entities are attached to a base entity at an activity.

 

The model used for this example, as well as all of the modeling "how-to" examples, can be found in the ProModel Solutions Café (http://www.promodel.com/solutionscafe/howto/). A video demonstrating this solution can be found here.

Solution

  1. For all entities routing to the activity to be assembled, select ATTACH as the type of routing in the Routing dialog for each. (The main or base entity to which the component entities are assembled can be of any routing type other than Attach.)
  2. Specify any time required to perform the assembly in the Time field of the activity, or click on the Logic in the activity Properties dialog and define a WAIT statement for the assembly time.

Tip: If the component parts are always available for the assembly, they don’t need to be modeled. Instead, just specify an activity time (see Activity Time How-To) representing the time it takes to perform the assembly operation.

Example

In the final assembly operation for a bicycle, two pedals and a seat are assembled to the bike.

Solution

Attach is selected as the type of routing in the Routing dialog. In the case of the pedals, the quantity is set to 2. Under the Quantity Mode , be sure to select Entire Quantity in the list.
