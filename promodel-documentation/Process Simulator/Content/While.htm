While

Statement

Syntax samples

While <Boolean expression> Do <statement block>

While Array1[n] <> 10 Do Inc n

While FreeCap(Loc1) > 5 Do
{

Inc Var2, 5
Wait 5 sec

}

Description

Repeats a statement or statement block continuously while a condition remains true. While is an entry-condition loop, meaning that the loop will not be executed once unless the Boolean expression is true.

Valid In

Any logic.

Components

<Boolean expression>

As long as this expression is True, the loop will continue. This expression is evaluated for each iteration of the loop.

<statement block>

The statement or block of statements to execute.

Example

The example below shows the logic for setting the values of Array1 elements 1 through 100 to the incremented value of the variable Count.

Int Count = 1
While Count <= 100 Do
{

CountArray1[Count] = Count
Inc Count

}
