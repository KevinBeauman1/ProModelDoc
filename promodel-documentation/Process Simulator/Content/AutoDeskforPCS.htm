Process Simulator - Autodesk® Edition

The Process Simulator Autodesk® Edition ribbon is provided in addition for Autodesk® subscribers for additional capability beyond building, simulating models, and review model results. The options include:

  • Asset Browser Allows the user to toggle on or off the Autodesk® Asset Browser within the Process Simulator environment inside Visio.
  • Sync Workflow Allows you sync your workflow to Inventor® or AutoCAD®. You can also export the workflow to file.
  • Pointer Tool Allows you to select the cursor pointer tool in Visio.
  • Connector Allows you to select the cursor connector option in Visio
  • Shapes Allows you to toggle on or off the Visio Shapes in addition to the Asset Browser.
