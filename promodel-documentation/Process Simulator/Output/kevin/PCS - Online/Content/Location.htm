Location

Function

Syntax samples

Location()

Attr1 = Location()
If Location() = ApprovalActivity Then Wait 4 min

Description

Returns the current activity’s name-index number. This function is especially useful in subroutines that vary depending on which activity’s logic calls them. By using a Location() function with an If Then statement, the subroutine can act differently depending on the activity that called it. Also, the same technique can be used to determine which activity is carrying out a process when multiple entities go through it.

Valid In

Activity, arrival and route logic.

Example

You may want to track the cycle time of a specific entity at each activity in its path through the process. To do this you need to add a check at the end of activity logic, for the entity's specific ObjectID and, if it matches, write out to an external text file the activity's name and the entity's cycle time.

If ObjectID() = 21 Then
{

aCycleTime = Clock() - aCycleTime
Writeline File1, "Activity: " $ Loc(Location()) $ ", " $ aCycleTime

}
