ScenarioName()

Returns current scenario name

Syntax samples

ScenarioName()

Description

Returns the Scenario Name when the function is invoked in logic.

Valid In

Valid in Most Logic.

Example

Suppose you choose to write out customer reports and want to know what Scenario you were running for the report, you would enter the following in your customer report logic:

ScenarioName()
