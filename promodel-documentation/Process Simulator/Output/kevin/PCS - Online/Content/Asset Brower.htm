Asset Browser

Selecting Asset Browser will open the Process Simulator Factory Asset Browser

Ribbon

  • Search - Allows user to search for named assets in the Asset Browser.
  • View button - Toggles Asset Browser view between Thumbnail and Tree view.
  • Sorting button - Allows user to sort assets in the Asset Browser.
  • Vault Access Button - Select Vault Access to establish or disconnect the secure connection to your Asset Vault server.

  • Refresh button - refreshes the asset view in the Asset Browser.

Shapes

The Asset Browser has 5 default Shapes:

  • Activity  The processing steps performed for each entity.
  • Buffer  The buffer shape provides an area where entities can accumulate and (optionally) be batched.
  • Entity  The items or people that are processed. Examples include documents, customers, parts, etc.
  • Resource  The agents used to move and process entities.
  • Storage  The storage shape provides an area where entities can be stored and then automatically replenished when triggered by a low quantity or after a certain amount of time.

Assets

All assets contained in the Process Simulator Asset Browser can be assigned one of six shape types: Activity, Entity, Resource, Buffer, Storage, or Plain Graphic (typically the assets will default to activity). Assets can be dragged and dropped to the layout to create a simulation model.

Assets are available in 4 repositories inside of the Asset Browser:

  • System Assets - Default assets available in Asset Browser.
  • User Assets - User assets on the users local drive.
  • Vault Assets - Assets available through an a secure vault server.
  • Cloud Assets - Assets available through the users Autodesk® cloud account.

Assets in the layout can be replaced by selecting the asset on the layout and then going to the asset you choose to replace it with and right click the new asset and choose to replace what is in the layout.

Sync Workflow

Assets may be exported to Inventor® or AutoCAD®. Syncing will auto lay-down assets in the respective Autodesk® software.

Within Inventor® or AutoCAD®, you can sync with Process Simulator from the Workflows addin Sync From Process Simulator. Process Simulator also allows you to save a Sync file which is used to sync the workflow if you choose to sync from within Inventor® or AutoCAD®.
