Options Dialog

The Options dialog is available from the Process Simulator ribbon and contains the following options:

  • Force Process Simulator connection style:  When you make a connection in Process Simulator, the connection uses a predetermined style. If you uncheck this option, connections can have any style you wish to define.
  • Immediate connection validation:  When you make a connection in Process Simulator, you can choose to be immediately notified if the connection is invalid by checking this option. If unchecked, the validation will occur when the simulation is started.
  • Add routing percentage value as text to newly created routing connections:  When you create a new Percentage routing connection, you may choose to have its percentage value automatically displayed on the connection.  Only when the second Percentage routing (or more) is created will the value be added to the connection.
  • Activity Time: When you add an activity to your diagram, this default time will be entered in the Activity Time shape property field. Right-click in this field to access the Distribution Builder for defining a distribution as your default activity time.
  • Move Time: With the Move Time field, you can set the default time that an entity takes to move to the next activity. Each newly created route connection will be given this default move time. Leaving this field blank will result in no move time during simulation.
