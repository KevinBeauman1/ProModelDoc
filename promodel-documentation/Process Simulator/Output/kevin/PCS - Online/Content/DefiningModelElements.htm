Defining Model Elements

Access to Attributes, Variables, Arrays, Macros, Subroutines, Resource Groups, External Arrivals, User Distributions, Initialization and Termination Logic, and External Files can be found in the Model Elements group on the Process Simulator ribbon and the More Element drop down list.

From these grids you may add, remove, and further define model elements in your model. You may also sort the columns by clicking their headers or reorder the list of these elements by using the up and down arrows in the grid toolbar.

Copying and Pasting in the Model Elements Grids

Copy and paste any field as well as full tables in all of the model elements grids.
