Assigning Raw Material Cost to an Entity

Assign a raw material cost to an entity, which is the initial cost of the entity as it begins processing.

 

The model used for this example, as well as all of the modeling "how-to" examples, can be found in the ProModel Solutions Café (http://www.promodel.com/solutionscafe/howto/).

Solution

  1. Open the entity dialog.
  2. Enter an initial entity cost.
  3. Each entity of this type that is created will automatically have this cost assigned to it.
Example

Castings are processed through an engine line to produce car engines. The initial cost of the casting is $78.

Solution

The initial cost of the Casting is set to $78.
