Pattern

Pattern type arrivals allow you to define your arrivals based upon a custom pattern where varying quantities of entities arrive within specified time buckets, similar to a histogram.

Entity  Shows the entity connected to the arrival connector and specifies the type of entities that will arrive. If an entity is not connected to the arrival connection, then this field will be enabled and the arrival entity is manually selected. 

Click the Define Arrivals button to open the Pattern Arrivals dialog.

How to define a pattern arrival

First, choose the arrival basis, Weekly Time or Calendar Date, by which you will define your pattern arrivals.

Next, click the Add button to add a pattern arrival record to the grid.

If the Pattern Arrival basis is set to Weekly Time, enter the Week number and week Day. If the basis is set to Calendar Date, enter the specific Date. The Start and End times define the duration over which the quantity of entities will be randomly distributed. Each row in the grid defines a unique duration and quantity, and all rows together define a custom pattern for the arriving entities.  A pattern arrival is most commonly used to model the arrival of people into a system like patients into a health clinic or customers arriving into a bank.  

If you use a Pattern arrival in your model, your simulation run length must be defined by either Weekly Time or Calendar Date, but not Time Only. The start time for each row must also be the same or later than the end time of the previous row. See Simulation Options for more information.

Additional options for the pattern arrival

  • Quantity Variability When toggled on, the quantities are entered into a Poisson distribution during simulation and thus given variability for the number of arriving entities. When toggled off, no variability is applied and the exact quantities specified are used.
  • Repeat Every  Allows you to choose whether or not to repeat the pattern defined and the frequency with which to repeat it. (Note: Repeat interval must be greater than or equal to the total pattern duration.)

Values for this grid can be copied and pasted to and from Microsoft Excel by right-clicking on a cell and selecting from the context menu. To copy data from the grid you must first select the desired cells to copy.  Once the desired cells are selected, right-click and select Copy from the context menu or use the Ctrl+C hot key to copy the data to the Windows clipboard.  The data can now be pasted into Excel.

When pasting data from Excel into the Pattern Arrivals grid, right-click and select Paste from the menu or use the Ctrl+V hot key to paste the data into the grid.
