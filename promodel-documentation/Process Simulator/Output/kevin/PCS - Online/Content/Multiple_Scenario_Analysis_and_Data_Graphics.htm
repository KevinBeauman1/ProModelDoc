Multiple Scenario Analysis and Data Graphics

This section will discuss utilizing Data Graphics when running multiple scenarios. In order to take advantage of Data Graphics and multiple scenario analysis, Process Simulator Data Graphics with the keyword ‘(Scenarios)’ in the name must be used.  Or, if custom data graphics have been created, the data graphic must use a Process Simulator data item containing a scenarios property.

  1. Launch the Scenario Manager.
  2. Create new scenarios, by selecting Add Scenario in the Scenario Manager.

  1. Place a check in the box for the scenarios you would like to run and compare.
  2. Select Run Scenarios.
  3. Apply a multiple scenario data graphic to your shapes in order to reflect the multiple scenario results that have been run.
