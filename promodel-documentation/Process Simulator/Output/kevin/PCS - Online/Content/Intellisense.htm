Intellisense

The logic for a model can be constructed with valid statements and expressions within a logic window. Logic windows possess a number of features that facilitate the writing of logic, and one such feature is Intellisense. Intellisense is a feature that becomes active when you start to type any character on a new line within a logic window. The Intellisense pop-up contains a list of all statements and functions, and of all previously created arrays, variables, macros, subroutines, attributes, entities, resources, etc.

When you begin a new line of logic, Intellisense list will appear and automatically filter elements that contain the characters that you have typed to that point. You need not type a full keyword or element name. The filtered list of elements contain the characters in the same order in which you typed them, but the characters are not necessarily contiguous in the listed elements. Intellisense will automatically highlight an element in the list. If this is the element you want, simply press the Enter, or Tab, or Spacebar key, and that element will automatically be entered into logic at the current cursor position. You can also use the arrow keys to select a different element in the Intellisense list prior to pressing Enter, Tab, or Spacebar to commit it.
