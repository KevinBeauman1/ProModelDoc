Conditional

  • Condition   The condition that must be satisfied, i.e. evaluate to true, before this route is selected. The condition can be an <Entity> or any variable/attribute value. For other special conditions, the test can be made in the activity logic, which sets an attribute value that can be tested here.
  • New Name   If you wish to have your entity renamed during routing with this conveyor, you may enter a new name to replace the old one. A new name cannot be entered if the condition is <Entity>.

Please note: If you have not defined any entities, variables or attributes in your process, the Conditional routing option will not be available. If none of the routing conditions evaluate to true during simulation, a run-time error will occur and terminate the simulation.
