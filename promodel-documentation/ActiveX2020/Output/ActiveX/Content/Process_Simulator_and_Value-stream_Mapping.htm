Process Simulator and Value Stream Mapping

Process Simulator is a perfect tool for creating and simulating value stream maps (VSMs). A VSM is essentially just a flowchart constructed using symbols that are specific to lean practitioners. Lean concepts include kanbans, supermarkets and takt times, all of which can be defined and simulated using Process Simulator to evaluate and improve process performance.

VSM Stencil

Process Simulator includes a value-stream mapping stencil that facilitates the creation and analysis of VSMs. This stencil can be opened by choosing it from More Shapes > Process Simulator > Value Stream Map. Once a VSM is defined, simulation properties can be added to the shapes through the standard Process Simulator properties windows. Both current state maps and future state maps can be defined and simulated.
