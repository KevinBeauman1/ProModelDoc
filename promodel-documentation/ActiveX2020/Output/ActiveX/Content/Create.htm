Create

  • When   Select whether to create the entity before or after the current activity.
  • Entity   Select the name of the entity to be created.
  • Quantity   The number of entities of the selected type to be created. This value may be entered as a number, or you can right click on the field to see a list of available variables, attributes, or other data elements, which may also be entered for the quantity.

Please note: If you have not defined any entities in your process, the Create routing option will not be available.
