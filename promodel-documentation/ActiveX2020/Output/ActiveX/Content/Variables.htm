Variables (Global)

Variables are of two types: global and local. Global variables are place holders defined by the user to represent changing numeric values. Local variables are place holders which are available only within the logic that declared them. Variables can contain either RealA real number is any number ranging from 1.7 X 10 -308 to 1.7 X 10 +308, including decimals. Real values may not include commas. As such, the number 5,380.5 should be entered as 5380.5. or IntegerAn integer number is a whole number ranging from -2,147,483,648 to 2,147,483,647. Integer values may not include commas. Therefore, the number 5,380 should be entered as 5380. numbers, and are typically used for decision making or recording information.

A global variable can be referenced anywhere numeric expressions are allowed in a model. If a variable is needed only in a single block of logic, it is easier to define a local variable right inside the logic block.

Global variables are defined in the Variables grid, and have the following options:

  • Name  The name of the variable. Double click on an existing name to change the name.

Variable names must be alpha-numeric, or may contain underscores. No other special characters are allowed such as %, $, !, etc.

  • Type   The type of variable, realA real number is any number ranging from 1.7 X 10 -308 to 1.7 X 10 +308, including decimals. Real values may not include commas. As such, the number 5,380.5 should be entered as 5380.5. or integerAn integer number is a whole number ranging from -2,147,483,648 to 2,147,483,647. Integer values may not include commas. Therefore, the number 5,380 should be entered as 5380..
  • Initial Value   The initial value of the variable to be assigned at the start of the simulation. By default, initial values are 0, but can be changed to whatever value you want.
  • Statistics   You can choose to collect no statistics, time-weighted statistics, or observation-based statistics for the variable.
  • Time-Weighted  Collects total changes, average minutes per change, current value, average value, and time-weighted statistics of the variable. The example below illustrates the calculation of time-weighted statistics for variables:

  • Graphic   The variable's graphic appears as a counter in the layout, and displays the change in the variable's value during simulation. If you delete the variable's graphic from the layout, the variable itself will not be deleted, the Graphic option will just become unselected. Select the Delete button in the grid toolbar while a record is highlighted in the list of variables to delete variables from the model.

Local variables are defined with Local Variable statements in Logic.
