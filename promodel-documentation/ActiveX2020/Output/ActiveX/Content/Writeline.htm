WriteLine

Statement

Syntax samples

Writeline <file ID>, <string or numeric expression>{,<maximum digits before decimal>, <digits after decimal>}

Description

Writes information to a text file and starts a new line. Writeline always appends to the file unless the file is Reset. Any file that is written to with Writeline automatically becomes a text file and will have an end of file marker attached to the end when it is closed.

Write and Writeline automatically separate values by commas

Valid In

Any logic.

Components

<file ID>

The name of the file as previously defined in the External Files grid.

<string or numeric expression>

The string or numeric expression to be written to the file.

<maximum digits before decimal>

When writing real numbers, the maximum number of digits before the decimal. Use this value to line numbers on different lines up on the decimal.

<digits after decimal>

When writing real numbers, the maximum number of digits after the decimal.

Please Note: The sum of the maximum digits before and after the decimal must be less than 20.

Example

The following example uses both Write and Writeline to record the time when EntA completes processing at Act1 in a write file called Rpt.

Wait N(7.3, .4)
Write Rpt, "EntA complete at:"
Writeline Rpt, Clock(min), 3, 2
