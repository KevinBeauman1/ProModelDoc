Tracking Per-use Costs for a Resource

Increase the unit cost of an entity by some constant value each time a particular resource is used.

 

The model used for this example, as well as all of the modeling "how-to" examples, can be found in the ProModel Solutions Café (http://www.promodel.com/solutionscafe/howto/).

Solution

  1. Open the resource dialog.
  2. Enter a cost per use for the resource.
  3. Each time during the simulation when this resource is used, the cost per use of the resource will be added to the unit cost of the entity.
Example

A dental X-ray machine is used at a dentist office that has a fixed charge of $50 per use. We want to add this cost to the patient cost.

Solution

The cost per use for the X-ray machine is set to $50. Alternatively, we could have defined the X-ray as an activity and entered IncCost (Entity, 50) to increment the entity cost by $50.
