Items

Items are the locationsactivities, resources, entities, variables, etc. available for displaying within a selected chart. Click the check box next to an item in the list and it will be added to the chart. Unselect a check box to remove an item from the chart.

Note:Pressing the Shift key while clicking on a checkbox will check or uncheck all items across all item groups. Pressing the Ctrl key while clicking on checkbox will check or uncheck all items within a group.

When you access a column chart an additional Statistics option becomes available under Items on the Filter Options panel.

In the Time Plot, Entity Count, and Resource Usage chart views, choose from "Pool By" options. The options are None and Scenario, with an additional Resource option for Resource Usage charts. In Resource Usage charts, you can also choose to display a Capacity Line to clearly view how much a resource is being utilized versus its capacity.

Column charts can only display one statistic at a time. Use the Statistic drop-down in the Filter Options panel to select which statistics to display in the column chart selected. In this example,% In Operation and % In Move Logic are shown. % Waiting and % Blocked are other options you can select.
