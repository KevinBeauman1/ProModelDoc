Options

Click the Options tab and the following ribbon appears.

Output Viewer makes it quick and easy to change what is displayed in your charts. For example, if you add or remove a scenario for a chart, it is also added or removed for all other charts in the Report. In essence, the Filter Options are linked between charts.

The Filter Tools - Options tab presents the options that affect whether the items in the Filter Options panel are linked when there is more than one chart in a Report. The options with checkboxes represent these common items. By default, scenarios for all charts in a Report are linked.

To link the Items list for one chart with other charts in the Report, check the "Items" option in the Options tab. To unlink a chart from others in the Report, first select a chart so it appears highlighted and then uncheck one or more of the Linked Filter Items options. To apply the same linking options to all charts in the Report, select the Synchronize Links button.

In the Report below there are two charts, LocationActivity Utilization and Single Capacity LocationActivity States. By default, the Scenarios option in the Filter Tools – Options tab is checked and therefore data about the Scenarios is displayed in both charts. In contrast, the Items option is unchecked, so the locationsactivities that both charts have in common, like “NC Lathe1” and “NC Lathe2”, are not linked. Notice that the Lathe locationsactivities are not displayed in the Utilization chart but are in the States chart.

To link the list of items common between the charts, select one of the charts to make it active, like the Single Capacity LocationActivity States chart. In the Filter Tools – Options tab, check the Items option and then click the Synchronize Links button. Notice in the screenshot below, this caused the Lathe locationsactivities to be displayed in the Utilization chart as well. Now when locationsactivities (common between the charts) are added or removed from one chart, they will be added or removed from both charts.

The Synchronize Links button makes it easy to link all charts together in a Report. Rather than changing the options for each chart, simply change the Linked Filter Items options for one chart, click the Synchronize Links button, and those options will be applied to all charts in the Report.
