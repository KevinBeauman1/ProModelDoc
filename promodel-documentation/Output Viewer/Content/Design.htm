Design

Click the Design tab and the following ribbon appears.

Note: This tab will not appear for tables and Gantt charts.

As your mouse hovers over the different Chart Type or Palette options, your active chart will show a live preview of how it will look if that chart type or palette is applied. The chart will not actually change to a new chart or palette until you click to apply it.

Change Chart Type

To easily interchange chart types, click on the Change Chart Type drop-down menu. Five categories of Chart types exist under the Change Chart Type drop-down: General, Bar, Column, Line and Other. Depending on what type of chart you are currently viewing, will dictate which Chart types will be available to select from the Change Chart Type drop-down menu.

For example, if a Pie chart is the active chart, and the Change ChartType drop-down is selected, a unique set of charts will appear. If a Time Series - Time Plot chart is the active chart and the Change Chart Type drop-down is selected, then a different unique set of chart types will be available to choose from.

As you hover over the different chart types under the Change Chart Type drop-down menu, a live preview of that chart is displayed in the Report View window. The chart will not actually change to a new type until you click to apply the type.

Change Palette

If the Share Style button is selected when you apply a new palette, all charts (except Table and Gantt) in all reports will assume that palette style.
