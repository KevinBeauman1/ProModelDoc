Charts

When you access Output Viewer directly from within a ProModel application, the Charts tab is the first ribbon tab you will see. After you run your simulation, Output Viewer automatically generates a default Report View of charts. You can create additional reports from the charts available via the Charts ribbon shown below.

Click on any chart in the ribbon and the chart and its associated filter options appear in the Report View.

  • Summary
    • Tables - more options provided
    • Column Charts- more options provided
  • Utilization
    • Location or Activity Utilization Chart
    • Resource Utilization Chart
  • State
    • Entity State Pie Chart
    • Location or Activity Single Capacity State Chart
    • Location or Activity Multi-Capacity State Chart
    • Resource State Chart
    • State Pie Charts- more options provided
  • Time Series
    • Time Plot
    • Histogram
    • Entity Count
    • Location or Activity State (over time)
