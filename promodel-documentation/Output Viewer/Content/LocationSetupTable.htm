Location Activity Setup Table

Click on the Tables drop-down and select Setup in the LocationActivity group.

A table is generated, which lists the percentage of time locationsactivities spent in setup, and contains the following columns of information.

Scenario: The scenario to which the data in the row applies.

Name: The name of the locationactivity where the setup downtime occurred.

Entity: The item that was setup at a locationan activity.

Total Setups:The number of times this locationactivity / entity combination resulted in a setup downtime.

Average Time Per Setup: The average time the locationactivity was down for each setup downtime.
