Logs Column Chart

This chart shows the summary data for log entries during simulation. Column charts can only display one statistic at a time. This example shows Average Value. Use the Statistics drop-down in the Filter Options panel to select other statistics.

Number of Observations: The number of log entries that occurred during the simulation for the given Log.

Minimum Value: The minimum log entry value during the simulation for the given Log.

Maximum Value: The maximum log entry value during the simulation for the given Log.

Average Value: The average value of all log entries during the simulation for the given Log.
