File

Click the File tab for file related options. Your Output Viewer must have one or more data sources loaded in order to provide charts and reports for you to view. The File tab allows you to start a new Output Viewer file (.pmov), open an existing one, or save Output Viewer files which contain charts and Report Views you create. It also provides access to loading additional output data source files into Output Viewer, for example, like output from a different model for results comparison. If you have opened the application from a simulated model to view results, any data sources associated with the model (e.g. scenarios) are automatically loaded into Output Viewer. If you have previously saved an Output Viewer file for your model, the file will be automatically loaded at the conclusion of your simulated run. If you launch Output Viewer by itself and not from another ProModel application, no data sources will be loaded. Follow these steps to load a data source.

If you launch Output Viewer from within a ProModel application before running a simulation, the following window appears.

From this window you can open Output Viewer files or add data sources to a new Output Viewer file.

How to Open and Save Output Viewer Files

  1. Select File > Open and an Open File dialog appears. Browse to a valid pmov file.

  1. Select the file and click Open to load the Output Viewer file.

  1. After you have created or modified some charts and reports, you can choose FIle > Save to save your changes.

How to Add Data Sources to an Existing Output Viewer File

You might want to add more data sources to an existing file to compare two different models' results or to compare results from several different iterations of a particular model. Different models are useful when different scenarios require changes beyond simple scenario parameter values.

  1. Click Add Data Source.

  1. The standard Windows file Open dialog appears.
  2. Select the RDB file you would like to add as your new data source.

  1. Once selected, it will be added to your existing data. The example below shows the addition of the Orders-Baseline Case.rdb to the Mfg. cost data.

How to Remove Data Sources

Hover over a loaded data source and a red [X] appears.

Click the red [X] to remove the data source from Output Viewer.
