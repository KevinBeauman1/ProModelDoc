Line

Use the Line option in the Format ribbon to adjust the weight of the Line of your Line chart elements. Time Plot - Time Series or any chart under the Line category from the Design - Chart Type - Line series must be selected for this option to be available.

Click up arrow [▲] in the Weight range box to increase the weight of the element line displayed in the chart.

Click down arrow [▼] in the Weight range box to decrease the weight of the element line displayed in the chart.

You can also type the desired weight in the Weight range box. In the example below, the line weight has been increased to 5.
