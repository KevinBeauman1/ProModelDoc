Location Activity Single Capacity State Chart

Click on LocationActivity Single Cap within the State group of the Charts ribbon.

The following chart is generated.

This chart shows a horizontal stacked bar chart summarizing the states of each single capacity locationactivity during the simulation run. States describe the conditions of these objects.

% In Operation: The percentage of time the locationactivity was processing an entity.

% Setup: The percentage of time the locationactivity spent in set up downtime during the simulation.

% Idle: The percentage of time a locationan activity was available, but not in use.

% Waiting: The percentage of time the locationactivity was waiting for a resource, or an entity, including Join, Load,attach actions and Match statements or a WAIT UNTIL condition in order to begin processing

% Blocked: The percentage of time the locationactivity spent waiting for a freed destination.

% Down: The percentage of time the locationactivity was not unavailable due to unscheduled downtime.
