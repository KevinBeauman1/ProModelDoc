Location Activity Summary Table

Click on the Tables drop-down and select Summary in the LocationActivity group.

A table is generated, which summarizes statistics gathered during the simulation that apply to all locationsactivities, and contains the following columns of information.

Scenario: The scenario to which the data in the row applies.

Name: The locationactivity name.

Scheduled Time: The total amount of time the locationactivity was scheduled to be available. (Excludes off-shift time, break time, and scheduled downtimes.)

Capacity: The capacity defined for this locationactivity in the model.

Total Entries: The total number of entities that entered the locationactivity, not including entities arriving to be joined and loadedattached. Entities split, unloaded, or ungroupeddetached or unbatched from another entity at a locationan activity do not count as additional entries. Arriving entities that have been previously grouped or loadedattached or batched to form a single entity only count as one entry.

Average Time Per Entry: The average time each entity spent at the locationactivity. This time may include partial times from the beginning and end of the actual run time.

Average Contents: The average number of entities at the locationactivity.

Maximum Contents: The maximum number of entities which occupied the locationactivity over the course of the simulation.

Current Contents: The number of entities remaining at the locationactivity when the simulation ended.

% Utilization: The percentage of capacity occupied, on average, during the simulation.

Cumulative Occupancy Time refers to the sum of the clock time each entity spends at a locationan activity for processing.
