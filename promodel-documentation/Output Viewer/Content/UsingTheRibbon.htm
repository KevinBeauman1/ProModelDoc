Using the Output Viewer Ribbon

When Output Viewer is launched, the following ribbon appears.

Click on the ribbon tabs: File, Charts, Export, Design, Layout, or Format to access the features and functionality offered in Output Viewer. Then click on the desired icon or button in the various ribbons to create charts, modify chart options, export data to Excel, take screen shots, or modify your chart's appearance.

The main application interface consists of the Ribbon, Filter Options panel, and Report View.  

There are three main tabs and three context-sensitive tabs in the Ribbon.

Main ribbon tabs:

  • File - Accesses Output Viewer files and data sources
  • Charts - Accesses all tables and charts
  • Export - Provides the ability to export charts in various formats

Context-sensitive ribbon tabs:

  • Design- Offers different color palette options for your charts
  • Layout - Offers options for changing the layout and appearance of your charts
  • Format (Charts) - Offers options for formatting the text of your chart

Different context-sensitive ribbon tab options are available based upon the chart that is selected as the active report. 

Note: If there are no charts selected in the active report, the context-sensitive ribbon tabs (Design, Layout, and Format) will not appear in the ribbon.
