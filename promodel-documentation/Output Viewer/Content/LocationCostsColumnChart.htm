Location Activity Costs Column Chart

This chart shows locationactivity cost data for each locationactivity collected during the simulation. Column charts can only display one statistic at a time.  This example shows Total Cost. Use the Statistics drop-down in the Filter Options panel to select other statistics to display.

Operation Cost: Operational Cost = (Active Operation Time * Rate) + (Any IncLocCost)

% Operation Cost: Refers to the locationactivity’s percentage of the sum of all operation costs

Resource Cost: Resource Cost = (Utilization * Rate) + (Times Used * Cost per use)

% Resource Cost: Refers to the locationactivity’s percentage of the sum of all resource costs

Total Cost: Total Cost = (Operation Cost + Resource Cost)

% Total Cost: Refers to locationactivity’s percentage of the sum of all locationactivity costs
