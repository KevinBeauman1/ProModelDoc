Creating and Modifying Reports

When you first access Output Viewer, both the Filter Options panel and Report View space are empty (unless it was launched after a simulation, or a Report View file (.pmov) was loaded). Click on a chart in the Charts ribbon and it will automatically be added to the active report, if empty, or to a new Report View. To add a chart next to an existing chart (or set of charts) in a report, click on a chart button in the Charts ribbon and drag the chart into the Report View of the existing chart.

Zoom mode

To quickly zoom in on any area of a chart, first make sure that your chart is selected by clicking on it. Next, while pressing the Ctrl+Shift keys, left-click in the chart area and drag the mouse. This creates the window frame in which you want to zoom into. When you have the size of the area you want to zoom into, let up on the left mouse button.

Zoom is also available from the Layout tab. For more information about zooming from the Layout tab, see Layout.

Adding New Reports

To add a new report to the Report View space, click on the [+] to the right of the last report tab and a new empty report appears. Then click on a chart button in the Charts ribbon that corresponds to a chart you would like to appear in the empty report.

You can also click on a chart button in the Charts ribbon and a new report is automatically generated with that chart in a new Report View.

Deleting Reports

To delete a Report View, click on the [x] located on the Report View tab.

Accessing Reports

To access an existing report, click on that report’s tab. You can also click on the down arrow [▼] located at the top right in the Report View space and a list of all reports will appear. Select a report.

You can also use the hot keys Ctrl+Tab to bring up a list of all Reports and cycle through each report by holding the Ctrl key while clicking the Tab key multiple times. To bring a Report into view, simply let go of the Ctrl and Tab keys.

Naming Reports

Double-click on a Report name in a Report View tab to open the Edit Title dialog. Then type the desired name for your report. You can also right-click on the Report View tab and choose Edit Title from the context menu to open the dialog.

You can also cut, copy and paste a report name in the Edit Title dialog. Double-click on the tab, then right-click in the Title field and the following context menu is available.

Maximizing Charts (Full Screen Mode)

Click the maximize button, located on the top right of the chart window title bar, or double-click the chart window title bar to expand the chart to full screen. To exit full screen mode, click the [x], double-click the chart window title bar, or press [ESC].

Note: If the chart is not maximized, clicking the [x] will remove the chart from the Report View.

Removing Charts from a Report View

To remove a chart from a report, click the [x] located on the top right of the chart window title bar.

Moving Charts within a Report View

When you have several charts in one Report View, to move or reposition a chart, click on the chart window title bar and drag the chart.

The chart will be temporarily removed from the Report View. Move it to a new position in the report by dragging and dropping it onto a placement marker.

Note: If the chart is dropped into an invalid space, the chart will return to its original placement in the Report View layout.

Moving Charts into a Different Report View

To move a chart from one Report View into another Report View, click on the chart window title bar and drag the chart.

The chart is temporarily removed from the Report View. Now drag the chart icon over the desired Report View tab. Hover over the tab until the Report View becomes active, continue to drag the chart icon into the new Report View and drop it on a placement marker.

Note: If the chart is dropped into an invalid space, the chart will return to its original placement in its original Report View.
