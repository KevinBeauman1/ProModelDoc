Location Activity Setup Column Chart

This chart shows setup data collected during the simulation. Column charts can only display one statistic at a time, this example shows Average Time Per Setup. Use the Statistics drop-down in the Filter Options panel to select other statistics to display.

Total Setups: The number of times this locationactivity / entity combination resulted in a setup downtime.

Average Time Per Setup: The average time the locationactivity was down for each setup downtime.
