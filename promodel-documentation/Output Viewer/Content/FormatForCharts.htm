Format (Charts)

Click the Format tab and the following ribbon appears.

Note: This tab does not appear for Gantt charts and for tables it provides different options.

Most charts will have these options in the Format ribbon. The options apply to the active chart and are enabled depending upon which option is selected in the Selected Item group.
