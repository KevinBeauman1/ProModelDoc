State

State Charts show horizontal stacked bar charts and pie charts of locationactivity, resource, and entity states. State Charts provide a summary view of the condition of these elements. This is useful for quickly seeing what your locationsactivities, resources, and entities spent their time doing over the course of the simulation. Such visualizations can help identify problem areas in your process. There are four state charts available:

  • Entity State Chart
  • Location or Activity Single Capacity State Chart
  • Location or Activity Multi-Capacity State Chart
  • Resource State Chart

There are four State Pie Charts available:

  • Entity State Pie Chart
  • Location or Activity Single Capacity State Pie Chart
  • Location or Activity Multi-Capacity State Pie Chart
  • Resource State Pie Chart
