Location Activity Costs Table

Click on the Tables drop-down and select Costs in the LocationActivity group.

A table is generated, which lists various costs for each of the model locationsactivities, and contains the following columns of information.

Scenario: The scenario to which the data in the row applies.

Name: The locationactivity name.

Operation Cost: Operation Cost = (Active Operation Time * Rate) + (Any IncLocCost)

% Operation Cost: Refers to the locationactivity’s percentage of the sum of all operation costs.

Resource Cost: Resource Cost = (Utilization * Rate) + (Times Used * Cost per Use)

% Resource Cost: Refers to the locationactivity’s percentage of the sum of all resource costs.

Total Cost:Total Cost = (Operation Cost + Resource Cost)

% Total Cost: Refers to the locationactivity’s percentage of the sum of all locationactivity costs.
