Networks Tab

The Networks tab shows the path networks in the animation, as well as each network’s path details.

  • Name: A list of names for each path network in the animation. Click on a name to view path information for that network in the right pane.
  • Color: Each network may have its own color. This is useful for visually identifying separate networks in the 3D layout.
  • Locked: When checked, locks the network in the 3D Layout so that it may not be accidentally moved.
  • Name (Path Details): Shows a list of the paths in the selected network. The names can be expanded to show each path segment’s points and their positions.
  • Position X/Y/Z: Sets the location, in layout units, for each joint on the path. For a description of layout units and their relation to the floor grid, see Layout Category on page 1.
