3D Animator Environment > Toolbar Options

Toolbar Options

The 3D Animator toolbar offers convenient access to many of the options available from the Menu bar.

All available toolbars are displayed in the image above:

  • File
  • 3D Layout
  • Animation
  • Edit
  • Setup
  • Objects

By default, the Setup and Objects toolbars are not visible. Using the View menu you may show or hide toolbars.

 

For information on using the icons in the toolbar, please see the descriptions of their corresponding options in the Menu bar, starting with Menu Options.
