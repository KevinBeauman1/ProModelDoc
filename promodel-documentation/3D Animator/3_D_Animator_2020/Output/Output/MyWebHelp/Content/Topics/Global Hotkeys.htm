Global hotkeys are typically used as a shortcut to access menu options. For example, each option in the Edit menu has a corresponding hotkey:

Please refer to each menu to view the available global hotkeys.

 

The only global hotkey not shown in a menu is F4, which will toggle the view between the Objects view and the 3D Layout view.
