Rotating Objects

Certain selectable objects can be rotated using the Object Manipulation tool. These include:

  • Locations graphics (not location counters).
  • Static resources.
  • Background graphics.

Using the Object Manipulation tool, hold the CTRL + SHIFT key and click and drag on one of the objects listed above to rotate it around the Z-Axis. You can also rotate around the X- and Y-axis by holding the X or Y(A) key in addition to Ctrl+Shift.
