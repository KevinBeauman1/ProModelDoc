Moving Objects only along the X-, Y-, or Z-Axis

Using the Object Manipulation tool, hold the X, Y(A), or Z key and click and drag on a selectable object in the 3D layout to move it along the corresponding axis.

 

The coordinates displayed in the Status Bar show where the object is being moved to.
