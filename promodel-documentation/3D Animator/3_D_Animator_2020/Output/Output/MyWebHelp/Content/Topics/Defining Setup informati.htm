Defining Setup information

Click on the setup navigation tab, which is located along the left side of the 3D Animator window.

The Setup navigation tab allows access to three categories of setup information, where you define initial settings, such as layout color, floor size, animation start time, and font information.

 

For more information on the option available from the Setup navigation tab, see Setup Navigation Tab
