Navigating Using the Walk-Through Mode

The walk-through mode places the camera at head-height, which emulates the perspective of walking along the floor of the layout.

 

To enter the walk-through mode, select the that option from the 3D Layout menu or the toolbar.

 

You may use the controls described in the “Fly Through Technique” to move the camera; however, the height will remain locked at head-height.

Please Note

Zooming, rotating, or changing views and perspectives will exit the walk-through mode.
