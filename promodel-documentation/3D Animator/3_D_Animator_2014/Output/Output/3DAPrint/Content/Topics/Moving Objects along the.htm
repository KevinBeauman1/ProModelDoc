Moving Objects along the X- and Y-Axes Simultaneously

Using the Object Manipulation tool, click and drag on a selectable object in the 3D layout to move it simultaneously along the X- and Y-Axes.

 

The coordinates displayed in the Status Bar show where the object is being moved to.
