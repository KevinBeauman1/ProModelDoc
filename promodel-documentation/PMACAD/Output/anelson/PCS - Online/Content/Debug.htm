Debug

Description

Brings up Process Simulator’s debugger. Use Debug to step through logic one statement at a time and examine variable and attribute values while developing a model. After a model is working, Debug statements are generally removed.

Valid In

Any logic.

Example

If you were having trouble with a Client’s logic at a particular activity, you could start the processing logic with a Debug statement, as in the following example. This would allow you to watch each Client’s logic execute, statement by statement, revealing problems with the logic’s flow.

Debug
If Attr1 = 1 Then

Inc ReworkVar

Wait 5 min
