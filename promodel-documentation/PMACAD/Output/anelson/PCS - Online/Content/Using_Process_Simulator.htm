Using Process Simulator

Using Process Simulator to model a process is a simple five-step procedure:

  1. Place Activity and Entity shapes on the Visio layout.
  2. Connect them to create a process flow of arrivals and routings.
  3. Fill out the information in the Shape Properties window for each of the shapes and connections.
  4. Run the simulation.
  5. Evaluate the output results.
