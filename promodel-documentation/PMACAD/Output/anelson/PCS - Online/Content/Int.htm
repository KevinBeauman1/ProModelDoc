Int

Statement

Syntax samples

Int <name1>{= <expression1>, <name2>= <expression2>...)

Int Count
Int Count = 1
Int Count = 1, Test = Rand(100)

Description

Creates a local variable of type integer. Local variables work much the same as attributes, except that they only are available within the logic that declares them. A new variable will be created for each entity that encounters an Int statement. Local variables are not directly available to subroutines, which have their own local variables. However, a local variable may be passed to a subroutine as a parameter.

Use local variables as the test variable in While and Do Until loops.

Valid In

Any logic. Variables declared with Int are valid in any expression within the logic where a normal integer number is valid.

Components

<names>

An identifier for the local variable. This identifier must be a valid name.

<expressions>

The variable will initially be assigned this value. This expression is evaluated every time the Int statement is encountered. If no expression is assigned, the local variable will be initialized as zero.

Example

A plant manufactures pipes of 12 different types, such as 2" steel, 4" aluminum, etc. At the Cutting station, these pipes are cut into certain lengths depending upon the orders that need fulfilling. As a pipe arrives at Cutting, it has an attribute aCutQty that specifies how many cuts are needed for that specific pipe. We can use a local variable to keep track of how many times the pipe is cut so that it can exit the While loop after completing the cuts. The logic might look like the following:

Int cutCount
While (cutCount <= attrMaxCuts) Do
{

Wait N(15, .4) sec

Inc varTotalPipeCuts
Inc cutCount

}
