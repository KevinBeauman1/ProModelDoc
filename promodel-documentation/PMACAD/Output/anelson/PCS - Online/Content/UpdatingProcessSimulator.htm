Updating Process Simulator

Periodically, ProModel releases updates to the Process Simulator program. If you have an Internet connection configured when running Process Simulator, Process Simulator automatically detects available updates, notifies you, with a prompts which will allow you the option to install them.

How to check for updates in Process Simulator

  1. Open Process Simulator.
  2. From the Process Simulator ribbon, under the sectionHelp you will see an icon that says Update Available when there is an update newer than the version you have installed.

Click the Update Available button to see the screen below:

Click the Update button. The update will begin to download.
