Installing Process Simulator

How to download, Install, and License Process Simulator from the URL link

Process Simulator works with Windows 10.

To download, install, and license Process Simulator, perform the following procedures:

  1. Confirm that Microsoft Visio is installed and registered on your computer. If you do not already have other prerequisite components installed, you will be alerted to install them during this installation process.
  2. Close all running applications.
  3. ProModel will have provided you with a URL link for downloading Process Simulator. Click on the URL link, or copy and paste the link into a web browser, and go to the URL web page.
  4. When prompted, choose to save the Process Simulator Setup file to your computer's Desktop.
  5. Double-click the file icon on the Desktop to initiate the installation. The installation will proceed according to the sequence below and may take several minutes.
  6. The Process Simulator Setup dialog box appears.

  1. Select the appropriate language, and click OK.
  2. The Process Simulator - Install window appears.
  3. When you click INSTALL, you implicitly agree to the associated license agreement. If you want to read the license agreement in its entirety, click the link By clicking Install, I agree to the terms of the license agreement on the dialog box.

  1. The Process Simulator - Setup Wizard message appears and displays the installation progress. The installation may take several minutes. No action is required on your part at this point.

  1. When the installation is complete, the Process Simulator - Finish dialog box appears.

  1. Click Finish.

You then can proceed to license the software. Click here for license instructions.
