Viewing the External Data Window

After running a simulation with the Visio External Data option checked in Simulation Properties, the External Data window should be automatically opened.  If the window was closed or a simulation was not run yet, the External Data window can still be opened.

  1. To view the External Data window, from the View menu make sure External Data Window is checked.

  1. The External Data Window appears below the layout and looks similar to a Microsoft Office Excel spread sheet.

Please note: If a simulation has not been run yet, no data will appear in the window.
