Using Data Graphics in Process Simulator

In order to use Data Graphics in Process Simulator you must first specify that you want to collect simulated results as Visio External Data.  Each time the model is run, the output results from that simulation will be reflected in the appropriate data graphics in the layout. To connect Visio External Data to the model, perform the following:

  1. Open the Simulation Properties dialog from the Process Simulator Ribbon, or the right-click menu on the layout.

  1. Check the Visio External Data checkbox under the Output viewer(s) to launch after simulation. By default, the Output Viewer checkbox is marked.

Please note: You can have more than one Output Viewer option checked, but in order to use Data Graphics the Visio External Data checkbox must be marked.

  1. Click OK.
  2. At this point data graphics can be applied to shapes in the layout.  However, because an external data link has not been established, no data will be displayed.  For more information on applying data graphics, see the section Applying Data Graphics to Shapes in the Model (see Applying Data Graphics to Shapes in the Model on page 1).
  3. Click Simulate to run the simulation.
  4. When the simulation is completed, select Yes to view the simulation results.
  5. The External Data window will be automatically opened and a dialog will be displayed showing the progress of auto-linking the simulated results to the shapes on the layout.
