Process Simulator User Interface Elements

With Visio open you will see two areas that are Process Simulator specific. There is also a right-click context menu that is not immediately visible:

  • Process Simulator Stencil Contains many shapes that you can use in your layout. The Process Simulator stencil does not need to remain open to use Process Simulator; you may use shapes from any stencil in your diagram. For more information, see Process Simulator Stencil.
  • Process Simulator Ribbon Has many tools needed for building and running Process Simulator models. For more information, see Process Simulator Ribbon.
  • Right-Click Menus Right-clicking on a shape or connection will show a context menu where you make changes to the shape or connection's properties. For more information, see Right-Click Menus.
