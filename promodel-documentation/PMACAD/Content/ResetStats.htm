Reset Stats

Statement

Syntax samples

Reset Stats

If Total = 20 Then Reset Stats

Description

Resets the simulation statistics. Useful in connection with the Report statement to manually control statistics for reporting purposes in case specific or event logic.

Valid In

Any logic.

Example

Suppose you want to generate a new output report for each 20-hour period of the simulation. Enter the following logic in an activated subroutine from Initialization logic:

While Clock(hr) < 10000 Do
{

Wait 20 hr
Report
Reset Stats

}
