FreeCap

Function

Syntax samples

FreeCap(<activity>)

Send FreeCap(Review) Form To Review

Description

Returns the available capacity of an activity. This function returns an integer.

Valid In

Any logic.

Components

<activity>

The name of the activity to examine. The Loc() function may also be used here.

Example

Suppose you are running low on inventory of Folder entities and you need to determine how many folders you need to order to fill it up. You could use the following logic:

If FreeCap(Bin) > 50 Then

Order FreeCap(Bin) Folder To Bin
