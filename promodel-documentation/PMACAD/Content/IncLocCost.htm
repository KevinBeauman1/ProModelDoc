IncLocCost

Statement

Syntax samples

IncLocCost <expression>

IncLocCost 15
IncLocCost -15
IncLocCost SetupCostVar

Description

Enables you to increment the cost (positively or negatively) of the current activity by a given amount. Use this function to add to the activity’s actual, accumulated dollar amount.

Valid In

Operation logic.

Components

<expression>

The positive or negative change to the value of cost.

Example

The logic below shows how to add a cost per entry to a location for an entering entity. Process Simulator automatically tracks operation cost per time unit specified in the cost dialog, however, you may assign a one time cost each time an entity enters a location.

In the following example, an entity, EntA, arrives for inspection at Inspect. We define no rate of operation cost for Inspect because each inspection accrues a one time cost regardless of how long the entity remains at Inspect. At Inspect, workers inspect EntA. After inspection, Process Simulator adds the cost to the activity through the IncLocCost statement and then to the entity with the IncEntCost statement.

Wait N(3, .7)
IncLocCost 3
IncEntCost 3
