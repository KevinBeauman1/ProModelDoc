Animate

Statement

Syntax samples

Animate <expression>

Animate 70
Animate Var1

Description

Sets the simulation’s animation speed. The higher the value, the faster the animation. Animate is used primarily to speed up or slow down a model for cursory or detailed observation whenever a particular condition is encountered. Another common use is to set the animation speed to one-hundred in the Initialization Logic to rapidly advance the simulation to some point in time.

Valid In

Any logic.

Components

<expression>

The new speed of the animation. This expression can be any number from 0 to 100 and is evaluated every time the Animate statement is encountered. The default speed is 56. A value of 0 will turn off animation.

Example

This example shows the use of the Animate statement in the logic for the activity Inspect. Whenever a part goes to Inspect for evaluation, the animation speed is slowed to 30 percent of maximum. This allows the modeler to more easily view which resource (Inspector or ProductionMgr) is captured to service the activity.

Animate 30.0
Get Inspector Or ProductionMgr
Wait N(5, .1)
Free All
