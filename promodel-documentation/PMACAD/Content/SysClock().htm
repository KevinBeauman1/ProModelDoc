SysClock()

Returns the system clock of the users PC

Syntax samples

SysClock

Description

Returns the System date and time when the function is invoked in logic.

Valid In

Valid in most Logic.

Example

Suppose you choose to write out customer reports and want to know what time each scenario or replication started simulating or stopped simulating for the report; you would enter the following in your customer report logic:

SysClock()
