Tank Logic Builder

An expanded capability within MedModelProModelServiceModel, the tank logic builder provides you with what you need to model complex tank and fluid system operations. The logic builder contains all available tank subroutines and provides you with a description of the components required to use each subroutine.

Please Note: The subroutine logic is not accessible until you define your first tank location—when you define the tank location, MedModelProModelServiceModel loads the tank submodel.

How to access the Logic Builder:

  1. Click the right mouse button in the logic window or expression edit field. Or click the Build button on the logic window’s toolbar.
  2. For more information about the Logic Builder, see Building the Logic on page 1.
