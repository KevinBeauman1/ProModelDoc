Network Client Installation Procedure

  1. Start Windows
  2. Insert CD-ROM or download the setup file from the Solutions Cafe to your desktop.
  3. While holding SHIFT on your keyboard, right click the installation exe and select "copy as path." This will copy the path to your clipboard.
  4. Open a CMD window by clicking on the windows key in the bottom left hand portion of your screen and typing cmd and select the cmd window option, or
  5. Select Accessories from All Programs:

 

 

  1. Select Command Prompt

  1. Paste the path you copied into the Command Prompt window. Right-click and select Paste.
  2. Press Spacebar and type the following: RUNTIMETYPE=Network.
  3. Press Enter and installation will begin.

Please Note: If the Setup Program did not open automatically, select Run... from the Windows Start menu. Type x:\install.exe (where x is the CD-ROM drive letter) and press ENTER. The CD Splash screen will appear. Then select Install.

  1. To start the installation, click OK. If any prerequisite components are needed, the Setup will install them for you. There may be a need to restart the machine. If this is the case, restart the installation by going back to step 3.

  1. Click Next to proceed with the installation.

  1. Select Finish. Upon running ProModel it will check your network for an available network license.
