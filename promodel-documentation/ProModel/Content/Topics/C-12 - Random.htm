Random

Syntax samples

RANDOM {<expression>}

Description

Randomly selects one of several available locations listed in a block of routings such that each location having available capacity is equally likely to be selected. If none of the locations listed has available capacity, the first location that becomes available will be selected.

Valid In

The rule field of the Routing edit table only.

Components

<expression>

The total entities output from the process. This expression is valid only for the first routing of a routing block. For more information on this expression, see Processing on page 1.

Example

Cars enter a parking lot and randomly park in one of N parking spaces. After EntA completes a 3 minute operation at Loc1, it is randomly routed to Loc2, Loc3, or Loc4.
