TimesUsed()

General System Function

Syntax samples

TimesUsed(<resource>)

If TimesUsed(Res1) > 5 Then Use Res2 for 10

Description

Returns the number of times a resource has been used.

Valid In

Any logic and any field evaluated after translation. For a list of fields evaluated after translation, see the “Appendix A” on page 1.

Components

<resource>

The name of the resource to examine. Res() may be used here.

Example

In the example below, customers (Cust) arrive at a local copy shop. Two resources (Trainee and Mgr) are available to service customers. If the trainee has serviced 10 or fewer clients, the manager must assist in helping the customer. Otherwise, the Trainee may help customers without the assistance of Mgr. In the example below, patients are seen by one of two doctors depending on how many patients each has seen. In this case, MD_B will be selected to see the patient only if MD_A has already seen at least five patients. In the example below, when an EntA arrives at Loc1, it will only use the resource Res1 if the resource has been used five or fewer times. If the resource has been used more than five times, EntA will use Res2 instead.
