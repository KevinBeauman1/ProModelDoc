Animate

General Action Statement

Syntax samples

Animate <expression>
Animate 70
Animate Var1

Description

Sets the simulation’s animation speed. The higher the value, the faster the animation. Animate is used primarily to speed up or slow down a model for cursory or detailed observation whenever a particular condition is encountered. Another common use is to set the animation speed to one-hundred in the Initialization Logic to rapidly advance the simulation to some point in time.

Valid In

Any logic.

Components

<expression>

The new speed of the animation. This expression can be any number from 0 to 100 and is evaluated every time the Animate statement is encountered. The default speed is 56. A value of 0 will disable animation.

Example

This example shows the use of the Animate statement in the downtime logic for the location Press. Whenever Press goes down for service, the animation speed is slowed to 30 percent of maximum. This allows the modeler to more easily view which resource (RES1 or RES2) is captured to service the location. This example shows the use of the Animate statement in the operation logic for the location X-ray. Whenever X-ray goes down for service, the animation speed is slowed to 30 percent of maximum. This allows the modeler to more easily view which resource (TECH1 or TECH2) is captured to service the location.
