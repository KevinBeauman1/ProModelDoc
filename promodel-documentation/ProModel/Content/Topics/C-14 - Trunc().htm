Trunc()

Type Conversion Function

Syntax samples

Trunc(<expression>)

Integer1=Trunc(3.9)

Description

Returns a real expression truncated to an integer. Any digits to the right of the decimal place will be removed. When necessary, MedModelProModelServiceModel automatically converts real values to integers by truncating them. For more information about MedModelProModelServiceModel automatically converting between reals and integers, see Converting Between Numeric Types on page 1.

Valid In

Any expression. This function returns an integer.

Components

<expression>

The expression to be truncated.
