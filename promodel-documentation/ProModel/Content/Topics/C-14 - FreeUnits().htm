FreeUnits()

General System Function

Syntax samples

FreeUnits(<location> or <resource>)

Use (FreeUnits(Res1)) Res1 For 5 min

Description

Returns the free units of a location or resource.

Valid In

Any logic and any field except those fields evaluated only at translation time. For a list of fields evaluated only at translation time, see Appendix A on page 1.

Components

<location>

The name of the location to examine.

<resource>

The name of the resource to examine.

Example

This example uses FreeUnits() to assign a team of Specialists to rework a problem. The size of the team, stored in the local variable T_Size, is determined by all the free Specialist units. Team_Time is a table function that varies the amount of time it takes to solve a problem (the amount of time in the Use statement) based on the number of units on the team. In the model of a day clinic, all staff members not actively engaged in providing patient care or performing some administrative task are called to a staff meeting. In the model below, an entity named “meeting” is sent to the conference room and all available staff members are gathered using the Get statement coupled with the FreeUnits expression.
