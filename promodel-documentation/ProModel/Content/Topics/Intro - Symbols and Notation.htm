Symbols and Notation

To better help you navigate this text, please review the following symbols and conventions.

Keyboard

The names of keys are displayed in capital letters. For example, ESC refers to the Escape key and CTRL refers to the Control key.

 

Keys are frequently specified in combinations or in a sequence of keystrokes. For example, CTRL + L means to hold down the CTRL key while pressing L. When key commands are set off by commas (e.g., ALT + N, R), press and release each of these keys (or key combinations) in the order listed. The term “arrow keys” refers collectively to the ñ, ò, ï, and ð cursor keys.

Text

Specific text you are asked to type is shown in bold type. For example, if you are directed to type cd mpsmod, you would type the lowercase letters “cd” followed by a space and the letters “mpsmod.”

 

Place holders for things such as file names and directories are shown in italics. For example, if you are directed to type filename.mod, enter the name of the file you wish to use (e.g., model_1.mod).

Logic

All MedModelProModelServiceModel reserved keywords are in uppercase.
