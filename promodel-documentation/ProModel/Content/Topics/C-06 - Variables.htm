Variables

Variables are of two types: global and local. Global variables are place holders defined by the user to represent changing numeric values. Local variables are place holders which are available only within the logic that declared them. Variables can contain either Real numbers or Integers, including element index values, and are typically used for decision making or recording information. A global variable can be referenced anywhere numeric expressions are allowed in a model. If a variable or attribute is needed only in a single block of logic, it is easier to define a local variable right inside the logic block. Global variables are defined in the Variables Editor, accessed from the Build menu. Local variables are defined with the Int and Real statements. See “Local Variables” on page 227 for more information.

How to access the variable edit table:

  1. Select Variables (global) from the Build menu.
