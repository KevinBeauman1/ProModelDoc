Attributes and the Join or Load Perm Statement

In some cases, one entity joins onto a base entity using the Join or Load Perm statement.See Join on page 1 for more information. If both the joining and base entities possess attributes before the join occurs, the resulting joined entity will possess the attribute values of the base entity.
