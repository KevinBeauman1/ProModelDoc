ProModel Cloud Upload

You can upload a model to ProModel Cloud. Follow these steps:

  1. Click File and select Upload Model to ProModel Cloud.

  1. The Upload dialog box appears.

  1. Enter the server URL where ProModel Cloud resides.
  2. Enter your valid Username and Password.
  3. Enter the name of the model you want to upload. It will automatically display the name of the model currently loaded.
  4. Update if model exists in the Repository — Select this box to update an existing model in ProModel Cloud. Both models must have the same name.

  1. Click Upload or Close.
