Attributes

Attributes are place holders similar to variables, but are attached to specific locations and entities and usually contain information about that location or entity. Attributes may contain integers or real numbers. You may also assign model element names (e.g., StationAExam_Rm1) to an attribute, which is stored as the element’s index number but may be referenced by name. Attributes are defined through the Build menu as shown below.

How to create and edit attributes:
