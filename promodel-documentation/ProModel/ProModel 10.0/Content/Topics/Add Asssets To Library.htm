Add Assets To Library

In the Autodesk® section of the ProModel Home Ribbon you can select Add Assets To Library. This will allow you to add Autodesk® assets to your model's graphic library.

Selecting Add Assets To Library will open the Add Assets Window where you can select from System, User, User Cloud, and Vault Assets. The assets can be selected once the asset image is visible by double clicking or using the center arrows to move selected assets to the right. Once the desired assets are selected, click OK to add the assets to the graphic library.

The Assets you select will then sync your Simulation model with Autodesk® AutoCAD® and Inventor®.
