Editing Process Records

When editing the records in the Processing edit table, two additional options for copying and pasting entire process and routing records appear in the right-click context menu.

The two additional menu items are as follows:

Copy Record Copies all fields of the current record for subsequent pasting.

Paste Record Places a copy of the most recently copied record above the current record.

Editing Process Record Notes: 1. A process record consists of a single row in the Process edit table, as well as one or more corresponding routing records defined in the Routing edit table for the given process. 2. From a Routing edit table, you may only use the Copy Record and Paste Record options to copy and create a duplicate routing record. 3. These options cannot be used to copy processing or routing records from one model to another.

How to duplicate Processing or Routing records:

  1. Select the desired record.
  2. Right-click in any field and from the context menu, select Copy Record.
  3. In the field you want to place the information, select Paste Record from the Edit menu (or press CTRL + V).
