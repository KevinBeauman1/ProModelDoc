View/Print Model Text

The modular nature of MedModelProModelServiceModel makes it easy to focus on the individual elements of a model. However, it can still be useful to see an entire model with all of the model elements in view at one time. MedModelProModelServiceModel provides two ways to accomplish this. Through the Export Model Text and the Print Layout options accessed from the Print button popout on the File application menu.

The Print section of the File application menu contains the following options.

Each menu selection is covered in detail in the following pages.

  • Print Layout Prints the model layout to a printer.
  • Print Model Text Prints the text of the current model to a printer.
  • Export Model Text Exports the text of the current model data to a file.
  • Printer Setup Opens a dialog box to allow printers to be selected and controlled.

Print Layout

You may print the layout of any model including all locations, path networks, resources, variables and background graphics to any printer configured for use with Windows. Regardless of the size of the model layout, the layout will be proportioned automatically to print on one standard size sheet of paper.

How to print a model layout:

  1. Select Print from the File application menu and choose Print Layout.
  2. Select the desired options from the Print dialog box and click Print.

  1. Choose the elements to be included in the layout.

Print Model Text

The Print Model Text and Print Layout options allow you to print a model to any printer configured for use with Windows.

How to print the current model text to a printer:

  1. Select Print Model Text from the options listed in the Print list accessed in the File application menu.
  2. Select the desired options from the Print dialog box and click PRINT.

Please Note: The entire layout may also be copied to the clipboard for editing and printing in another application.

Export Model Text

The Export Model Text option saves the text of the current model data to a file, which can be viewed and searched in an external text editor program.

How to export the text of a model:

  • Select Export Model Text from the options listed in the Print list accessed in the File application menu. After the model text is exported, you are prompted if you would like to view the exported text.

Only the first 30 characters of names will appear in the names column of the exported model text.

Printer Setup

MedModel ProModel ServiceModel allows you to print to any printer configured for use with Windows. At times you may need to switch from the default printer to another printer. This can be done easily through the Setup option on the print dialog box.

How to change the printer settings:

  1. Select Printer Setup from the Print list accessed in theFile application menu to access the Print Setup dialog box.
  2. Select the desired options and click OK.
