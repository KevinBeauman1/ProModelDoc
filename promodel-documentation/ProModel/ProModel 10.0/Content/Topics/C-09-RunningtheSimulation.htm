Simulation Start

Choosing Simulate begins the simulation of the current model. The model is automatically saved in a file called autosave.mod. Therefore, if the simulation is terminated abnormally, MedModelProModelServiceModel will ask you if you would like to load the latest autosave.mod file when you re-open MedModelProModelServiceModel.

Once you choose to Simulate, a translation status window appears, showing which data is currently being translated. This gives you the option to abort the simulation run at any time.

Abort Allows you to cancel model translation at any time during translation.

Continue Allows you to continue model translation when a warning message is displayed.

Detailed Status Allows you to view more specific information about the data in translation.
