Settings

The following table defines each of the selections available from the settings section of the View menu. These selections and their submenus are discussed in more detail on the following pages.

Views Allows you to define, then quickly and easily access specific areas of the model layout. Once the view is defined, you can select it while editing or running the simulation. The various View settings are further addressed in Views on page 1.

 

Zoom to Fit Layout Shrinks or enlarges the layout to include the entire model.

 

Zoom Allows you to shrink or enlarge the layout by the percentage selected.
