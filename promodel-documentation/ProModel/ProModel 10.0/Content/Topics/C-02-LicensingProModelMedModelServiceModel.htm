Licensing ProModelMedModel

Your ProModelMedModel product must be licensed. If you do not license ProModelMedModel, you will only have access to a limited version of ProModelMedModel which will not allow model building.

If you purchased License, you will log into the software after the installation process.

How to license ProModelMedModel

1. After installing, start the application.

2. A log in prompt should appear. Choose Sign Up to create your new ProModel account. You can also choose to authenticate with an Autodesk User Account for convenience. If you have previously signed up for an account, select Log In.

3. Create a password for your new ProModel account.

4. Click Sign Up and then the following message should appear.

5. Go to your email program to get the Verify Email sent to you.

6. Click on the Verify My Email button in the email and it should open a web page with the following verification message.

7. Now return to the ProModel application and click the Login message by clicking the following button in the ribbon.

8. This will open the login screen again. Enter your email address, newly created password and then click the Log In button.

9. You have created a ProModel account. Next you will connect this account with the new ProModel licensing system. So you will get the following prompt.

10. Click the Request Code button and you should receive this message:

11. Click OK and then go to your email and get the Auth Code that was sent to you.

12. Copy the Auth Code and then enter it into the following dialog.

13. Click OK and you should now be successfully signed into the application!

As you use ProModelMedModel it may become necessary to transfer your License. Please contact ProModel Corporation technical support if you would like to move or terminate your software key.
