Ln()

Math Function

Syntax samples

Ln(<expression>)

Real1 = Ln(Real2)

Description

Returns the natural logarithm of an expression.

Valid In

Any expression. This function returns a real number.

Components

<expression>

Ln() returns the natural logarithm of this expression.

See Also

Exp().

Please Note: To get a logarithm to a particular base, use the following formula: logbase<expression> = Ln<expression>/Ln<base>
