File Menu

The File application menu is the first tab on the ribbon bar and consists of six sections divided by separator lines along with a listing of the most recently loaded models. Information about the currently loaded model is the first section in the menu. The file management section contains functions related to model files such as opening and saving. The print section allows you to export a text listing of the current model and print that listing or model layout. The packaging section allows you to create and install model packages containing models and their associated files. The Options section provides access to application level settings and Exit closes MedModelProModel. The model history section lists the most recently opened models for quick retrieval. Choosing any model in the model history will load that model.
