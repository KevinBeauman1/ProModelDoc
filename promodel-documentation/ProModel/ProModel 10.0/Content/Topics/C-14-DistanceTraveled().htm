DistanceTraveled()

Returns distance traveled of a resource and its unit.

Syntax samples

DistanceTraveled(<resource name> {,unit})

Description

Returns the distance traveled of a resource; and its unit/s.

Valid In

Valid in Most Logic a resource can be used.

Example

An AGV requires charging after it has traveled X distance. DistanceTraveled() can be used to capture that distance for a resource or each unit of the resource.

inc vChargeRes1Unit1, DistanceTraveled(Res1, 1)
