Tools Tab

The Tools ribbon tab gives you access to powerful tools to help you through the model building process.

Calendar Editor Shifts and time off for activities and resources are defined using the Calendar Editor.

Graphic Editor Allows you to create, edit, rearrange, or delete library graphics for use as entities, locations, resources and background graphics.

3D Animator Launches 3D Animator, which can create a 3D model and animation of any models you create in MedModelProModel.

Model Collaborator Launches Model Collaborator, which allows you to combine two or more models into a single model.

Multi-View Runner Launches Multi-View Runner, which allows you to run a simulation with multiple synchronized model views simultaneously.

License Manager Launches the license manager. Since license changes cannot be made while MedModelProModelServiceModel is running, you will be prompted to close MedModelProModelServiceModel prior to running the License Manager.
