Defining Attributes

The Attributes module allows you to define an Entity Attribute that can be used to tag entities with unique values. For example, we can assign an attribute the value of one if a part passes inspection, or to zero if it fails the inspection. This can be useful for determine what to do with the part later in the process. For more information about defining attributes, see Attributes on page 1.
