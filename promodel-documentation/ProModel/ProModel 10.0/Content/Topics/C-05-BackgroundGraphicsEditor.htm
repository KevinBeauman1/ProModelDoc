Background Graphics Editor

The Background Graphics Editor allows you to place icons, text and other graphic shapes on the layout behind locations and other system element graphics. The default arrangement of the two windows is shown below.

  • The Library Graphics window, containing all the icons in the current graphic library, is located at the left of the workspace by default.
  • The Layout window, where all creating and editing of graphic shapes is done, is located in the right portion of the workspace.
  • The Insert ribbon tab, contains shapes, text, and access to external images that can be inserted into the background.

Library Graphics Window

The Library Graphics window contains the icons of the current graphic library file, specified in the General Information dialog. These icons may be placed on the Layout in the same way as other objects.

You may size the window as desired, or use the scroll bar shown above to scroll through the available icons.

Insert Ribbon

The Insert ribbon contains tools necessary to create background graphics of various shapes, including text boxes and images created in other graphics programs. For more information, see Insert Tab on page 1.

Importing a Graphic

Importing a background graphic can bring reality into the model. For example, if a layout is created in a graphics program, it may be desirable to import the entire layout rather than create it in MedModelProModelServiceModel. This is done by saving the file in a graphics program, such as AutoCAD, as a DWG (with the correct layers selected, or PNG, JPEG, BMP, PCX, or GIF file and importing the graphic into MedModelProModelServiceModel. This can save you an extensive amount of time.

How to import a background graphic to scale into the layout:

  1. Select the Insert ribbon tab.
  2. Select the Background Graphic to Scale button.
  3. Enter the name of the graphic you would like to import or browse to the directory where the graphic is located.
  4. Select OK to close the import graphic dialog box. The graphic will then appear in the Background Graphic to Scale window.
  5. With your mouse you can click and drag the image in the preview window; also using your scroll wheel to zoom in and out of the graphic to find the desired section that you will use to scale the image.
  6. Once you have the section of the image in the view window, you select the st needs to click the "Place Marker" button and click on two points within their graphic. Marker points should not be able to be placed outside the graphic. Control should only allow vertical or horizontal lines to be drawn.

    2. Once both markers have been placed additional text box options will show up:
    3. Distance: this is the distance in ft. or meters (as defined in File/Info) that is between the two marker points. This must be a whole or real number greater than 0.
    4. # of grid units: this is the number of grid units they want to fit within the two markers they placed. This must be a whole number greater than 0.
    5. The upper left corner of the imported graphic will align with the upper left corner of the layout window.

The purpose of this feature is to import a background graphic and scale the ProModel grid according to a known "real life" distance depicted in the imported image and fit a defined number of grid units within that distance.:

How to size an imported background graphic:

  1. Place the cursor over one of the four corners of the imported background graphic. The cursor becomes a cross-hair at this point.
  2. Click and drag the cursor to size the background graphic as desired.

Please Note: Once imported, the background graphic is not a separate file from the model. It is included in the model. Therefore, when moving or copying a model file from one directory to another, it is not necessary to move or copy the imported background graphic file as well. On the other hand, if the external graphic file is changed, it must be re-imported to update the model layout.

Right-Click Context Menu

The Graphics Menu allows you to flip and rotate the selected graphic object(s) in the layout window. It also allows you to specify whether you want the graphic to be behind or in front of the grid. Additionally, it allows you to group selected graphic objects together into a single graphic. Finally, it provides the option to define line styles, fill patterns, line color, and fill color.

Flip Horizontal Horizontally flips the selected object(s).

Flip Vertical Vertically flips the selected object(s).

Rotate Rotates the selected object(s) 90 degrees clockwise. This does not apply for non-true-type fonts.

Bring to Front Gives the option to bring the selected object to the very top of all graphics or bring it forward one position at a time.

Send to Back Gives the option to send the selected object to the very bottom of all graphics or send it backward one position at a time.

Behind Grid Toggles the selected graphic(s) to show in front of or behind the grid.

Group Gives options for grouping or ungrouping several graphics. Combine or group several graphic objects into a single graphic so they may be sized and edited together. Ungroup several grouped graphic objects so they may be edited individually.

Lock Locks a graphic in place so that it can’t be moved. This is helpful for preventing accidental moving of a graphic that you wish to leave stationary.

Alignment When multiple objects are selected on the layout, they can be aligned side to side, top to top, etc.

Line Styles Allows the user to define the line style including transparent, dashed, line thickness, and optional arrowheads on either end of the line. If any objects are selected, the line styles of the selected objects are changed.

Fill Patterns Allows the user to define the fill pattern for solid objects including slant, grid, crosshatch, backward slant, horizontal, vertical, transparent, solid, vertical gradient, and horizontal gradient. If any objects are selected, the fill patterns of the selected objects are changed.

Line Color Allows the user to define the line color and create custom colors. If any objects are selected, the line color of the selected objects are changed.

Fill Color Allows the user to define the fill color and create custom colors for solid objects. If any objects are selected, the fill color of the selected objects are changed.

How to set a graphic to show behind the grid:

  1. Select the graphic on the layout using the selector.
  2. Right-click on the graphic and select Behind Grid from the context menu.
