Modeling Environment

The Modeling Environment is everything contained within the MedModelProModel window.

When you open a model or select New from the File menu, your screen appears with a Ribbon bar across the top of the screen and a Layout window. You will also be given access to the MedModelProModel Getting Started Panel. For now, let's look briefly at the Getting Started Panel and then the menus accessible from the menu bar.

From the MedModelProModel Getting Started panel, you may quickly access some of MedModelProModel’s resources to help you learn how to use the application and model different situations:

  • Demos Opens a window with a list of various demonstration models to explore and simulate.
  • Quickstart Video Loads a video showing you how to build a model.
  • How To Videos Loads a web page containing many videos showing how to model various real life situations.
  • Training Resources Loads a web page containing other sources for learning how to use the program.
  • Help Launches the online help system.
  • Support Provides the contact information for Technical Support

To simply begin working on a new model, close the Getting Started Panel and select New from the File application menu. The Getting Started Panel can be opened again from the View ribbon tab.
