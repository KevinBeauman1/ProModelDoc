How to create a model package:

  1. Select Create Model Package from the Package button in the File application menu.

  1. Enter the name you wish to use for the model package (by default, MedModelProModelServiceModel uses the name of the current model with a *.pkg extension). You may also use the Browse... button to select the model name and directory.
  2. Check the Exclude Graphics Library box if you want to package the model without the graphics library.
  3. The Re-save model check box will be checked by default.
  4. Click OK.
