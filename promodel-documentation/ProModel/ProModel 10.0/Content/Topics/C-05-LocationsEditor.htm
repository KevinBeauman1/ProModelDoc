Locations Editor

The Locations Editor consists of three main windows: the Location Graphics window in the lower left portion of the screen, the Locations edit table along the top of the screen, and the Layout window in the lower-right portion of the screen. These windows can be moved, docked and resized using the mouse.

Sub-windows such as Clock Downtime and its associated Logic window can also be docked within the application workspace. The Locations edit table contains information about every location in the model, including characteristics such as capacity and number of units. The Location Graphics window is a tool box used for creating, editing, and deleting locations graphically. Locations are positioned in the Layout window.
