Model Instructions

The Model Instructions window displays background information about a previously constructed model that has been loaded into the MedModelProModel program. The information can be helpful in using demonstration models or models created by ProModel's consultants. If a file containing the background information in Rich Text Format (RTF) was previously specified in the Model Instructions box of the General Information dialog, the Model Instructions window immediately appears when a model is loaded. For information about the General Information dialog, refer to the topic General Information Dialog Box.

The Model Instructions window has two buttons, Simulate Model and Explore Model.

  • Click Simulate Model to initiate a simulation of model scenarios.
  • Click Explore Model to close the Model Instructions window and explore the logic and other elements of the loaded model.

How to manually open the Model Instructions window:

  • Select Model Instructions from the View ribbon tab.
