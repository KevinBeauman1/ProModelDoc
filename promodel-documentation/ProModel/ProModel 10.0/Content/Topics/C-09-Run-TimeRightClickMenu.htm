Right-Click Context Menu

Right clicking in the simulation window will bring up a right-click context menu, which gives you easy access to several animation controls.

Animation Off Turns off the animation, which makes the simulation run faster.

Zoom to Fit Layout Brings all model graphics into view.

Zoom Allows you to choose a level of zoom.

Views Allows you to choose a view that you have previously defined. See Views on page 1 for information on defining views.

End Simulation Allows you to interupt and end the simulation during run-time.

Pause/Resume Simulation Allows you to toggle the simulation running or pausing.

Trace Options Lists events as they happen during a simulation. This listing may be Step by Step or Continuous.

Dynamic Plots Opens the Dynamic Plots window.

Please Note: Right clicking to bring up the context menu will pause the simulation as long as it remains open.
