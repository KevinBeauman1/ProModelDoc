Zoom Options

Zoom options are accessible during both edit and run time. In edit mode, Zoom is found in the View ribbon tab. The Zoom List dropdown gives several options for zooming including a custom zoom.

Zoom to Fit Shrinks or enlarges the contents in the layout window to show the entire model.

Zoom Allows you to shrink or enlarge the layout by the percentage selected or based on the custom zoom value entered.

How to zoom in or out on the layout:

  1. Select Zoom from the View ribbon tab.
  2. Choose a percentage from the list or in the Custom Zoom text box, type a number for the percentage to zoom the layout.

Please Note: The minimum and maximum zoom levels are calculated automatically depending on the total size of the layout at 100% zoom.
