Defining Arrivals

Of all the entity types, only some need to be scheduled to arrive in the system. Every 12 minutes one Cart arrives at the AltRoll location. The word “INF” in the Occurrences column means that one Cart continues to arrive every 12 minutes as long as the simulation runs. For more information on defining arrivals, see Arrivals on page 1.

Defining Arrivals

According to our records, patients arrive every 45 minutes. Therefore, frequency (i.e., time between arrivals) is set to 45. Since the default time unit for the model was set to minutes, this entry will produce arrivals every 45 minutes—exactly. To take a more probabilistic approach, you can create arrivals according to an exponential distribution with a mean of 45 minutes by entering E(45).
