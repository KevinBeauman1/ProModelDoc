In the processing logic above you can see how each customer’s Quantity attribute is set to the value returned from a user-defined distribution. This value is then multiplied by the unit time per copy to obtain the total processing time. For self-serve customers, the time per copy is 3 seconds. For operator-assisted customers, the time per copy is .01 minutes (0.6 seconds).

 

In addition to the new processing times based on order size, the Match command has been implemented at both the order desk and at the high-speed copy machines to ensure that the proper order is returned to the customer who placed the order.
