Sqrt()

Math Function

Syntax samples

Sqrt(<expression>)

Real1 = Sqrt(Real2)

Description

Returns the square root of an expression.

Valid In

Any expression. This function returns a real number.

Components

<expression>

Sqrt() returns the square root of this expression.

See Also

Round().

Please Note: To get a root other than the square root, use the exponentiation operator as shown in the following formula:

X**(1/Y)

For example, where Y is the desired root, the formula 9**(1/3) returns the cube root of 9.
