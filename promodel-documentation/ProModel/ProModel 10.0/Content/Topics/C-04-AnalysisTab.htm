Analysis Tab

The Analysis ribbon tab provides options for performing various analyses on your model and integrating historical data into your model. You can take data from your actual system you are modeling and fit it to custom or standard statistical distributions, optimize your model given a range of inputs and a user-defined objective function, and perform indepth six sigma statistical analysis on the model output.

The selections from the Analysis Tab are as follows:

Streams Takes you to the Streams module for defining and manipulating the random number streams. For more information, see Streams on page 1

Table Functions Opens the Table Functions module for defining data look up tables. For more information, see Table Functions on page 1

Cost Opens the Cost definition window for adding values for activity-based cost analysis. For more information, see Cost on page 1

Context Takes you to the User Distributions module for defining empirical data functions. For more information, see User Defined Distributions on page 1

Output Viewer Launches the program for viewing and analyzing results generated by your model. For more information, see Output Viewer on page 1

Minitab Launches Minitab for six sigma statistical analysis. For more information, see Viewing Results in Minitab on page 1

Six Sigma Opens the Six Sigma Configuration dialog for setting up which statistics should be included in your analysis in Minitab. For more information, see Six Sigma Configuration Dialog on page 1

SimRunner Launches the SimRunner Optimization program, which allows you to run an optimization analysis on your model. For more information, see SimRunner on page 1

Stat::Fit Launches the Stat::Fit statistical curve fitting program.
