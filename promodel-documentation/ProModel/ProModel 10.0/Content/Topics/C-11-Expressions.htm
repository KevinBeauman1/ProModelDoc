Expressions

Expressions consist of a value or combination of values (even of different types), variables, attributes, functions, and operators that result in a value. Many consider an expression to be a string expression if it evaluates to a string, a numeric expression if it evaluates to a number, and so on. (For information on how to handle different value types in the same expression, see Converting Between Numeric Types on page 1.)
