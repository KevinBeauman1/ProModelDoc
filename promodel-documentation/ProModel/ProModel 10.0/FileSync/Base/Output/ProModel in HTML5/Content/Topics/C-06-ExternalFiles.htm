External Files

External files may be used during the simulation to read data into the simulation or write data as output from the simulation. Files can also be used to specify such things as operation times, arrival schedules, shift schedules, and external subroutines. All external files used with a model must be listed in the External Files Editor which is accessed from the Home ribbon tab.

How to define external files:

  1. Select External Files from the Home ribbon.
