Processing

Processing defines the routing of entities through the system and the operations that take place at each location they enter. Once entities have entered the system, as defined in the Arrivals table, processing specifies everything that happens to them until they exit the system.

Processing is defined in the Processing Editor, which is accessed from the Home ribbon tab. This section first describes how to create simple processes, then explains each feature of the Processing Editor.

How to create and edit process routings:

  • Select Processing from the Home ribbon tab.

or...

  • Right click on the existing process routing and select Edit Routing.
