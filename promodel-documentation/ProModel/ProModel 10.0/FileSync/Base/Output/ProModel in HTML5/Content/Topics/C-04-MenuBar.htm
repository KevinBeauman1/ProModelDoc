Ribbon Bar

All of the tools necessary to build and run a model and view the corresponding output are accessed through the ribbon bar. The ribbon bar is located just beneath the ProModel title bar and contains the tabs listed on the following page. These ribbon tabs access other items with functionality related to the tab heading.

  • File The File application menu allows you to open new models, save current models, and create or install model packages. It gives you access to general information and settings for the loaded model as well as application options. It also allows you to export a text version of the model and print either the model text or the graphic layout of the model. For more information, see File Menu.
  • Home The Home tab contains most of the modules for creating and editing a model. This includes the basic modules such as Locations, Entities, Arrivals and Processing, and the modeling elements such as Variables, Attributes, Arrays, and Subroutines. For more information, see Home Tab.
  • Insert The Insert tab lets you add text, pictures, and various shapes to your model layout. It also allows you to merge other models into the existing one. For more information, see Insert Tab.
  • Analysis The Analysis tab provides options for performing various analyses on your model and integrating historical data into your model. For more information, see Analysis Tab.
  • View The View tab lets you manipulate the model Layout view. From this tab you can control layout settings, hide or view hidden paths, operate the zoom controls, and more. For more information, see View Tab.
  • Tools The Tools tab contains various utilities including the Graphics Editor for creating and modifying graphic icons and the Calendar Editor for defining shifts and time off. For more information, see Tools Tab.
