Network Installation Procedures

Overview

ProModel Corporation offers a network edition of ProModel that allows a single machine to control license usage (via a network connection) for several users. To use this method of license tracking, you must install a network edition of ProModel. The network edition includes the ProModel client program, a hardware security key and the license manager application you will use to control license usage at your site.

 

The network edition consists of the following:

 

  • Network License Server Setup
  • Network Client Installation

 

When you install and run the network client edition of ProModel, it searches your network for a license key server, rather than checking its own USB port for a security key. Once it locates the license key server, ProModel asks the license manager application (running on the key server) if a license is available. If a license is available, the license manager will assign a license to the client computer. When you exit ProModel, the license assigned to you is freed and becomes available to another user.

 

The license server consists of the security key that plugs into the license server’s USB port, and the license manager software. When installed, the license server responds to ProModel client license requests and allows computers to run either a full version (if sufficient licenses are available) or a limited run-time version (if no license is available). The license server can run on Windows 10, 8.1 or 7 and Windows Server 2008 and listens for incoming license requests over any of the following network protocols: IPX, TCP/IP, and NetBEUI. If the license server is not functioning, ProModel will start as a limited run-time version.

 

Check the support web page for periodic updates to the installation procedures, software versions, and the license server components.
