Standard Installation Procedure

  1. Start Windows.
  2. Insert the CD-ROM or download the setup file from the Solutions Cafe.
  3. Run the installation by selecting Install from the CD splash screen or double-clicking on the MedModel Professional.exe file.
  4. Click OK. If any prerequisite components are needed, the setup will install them for you. There may be a need to restart the computer. Restart the installation each time a reboot needs to occur.
Please Note: If the Setup Program did not open automatically, select Run... from the Windows Start menu. Type x:\install.exe (x is the CD-ROM drive letter) and press ENTER. The CD splash screen appears. Select Install.
  1. Select the language you would prefer to use. Click OK.

  1. The MedModel Setup will begin.

  1. When you click the INSTALL button in the MedModel - Install window, you implicitly agree to the associated license agreement. If you want to read the license agreement in its entirety, click the link By clicking Install, I agree to the terms of the license agreement in the window.

  1. The MedModel installation will continue and displays the setup progress. The installation may take several minutes. No action is required on your part at this point.

  1. Click Finish.
