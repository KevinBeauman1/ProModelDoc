AutoCAD Plug-in

The ProModel AutoCAD® Edition's AutoCAD® plug-in is a powerful tool that allows you to give your CAD drawing (DWG file) ProModel simulation properties.

The ProModel AutoCAD® Edition Plug-in provides a ribbon tab that provides quick access for developing your model and opening it in ProModel. As you hover your mouse over the buttons in the ribbon, tool tips will appear describing the module as well as showing hotkeys for quick keyboard access to the module.

The selections from the AutoCAD® Edition Home Tab are as follows:

Add Process Flow Quickly add a process flow to your AutoCAD® DWG file.

Simulation Browser Opens ProModel's Simulation Browser to browse or modify the model elements of the DWG file.

Simulation Properties Opens the ProModel Simulation Properties for each of the model elements the user added to the DWG file.

Open In ProModel Open the simulation in ProModel to simulate and add additional modeling elements to the system being modeled.

Online Help Opens the ProModel AutoCAD® Edition's Online Help.
