Background Graphic to Scale

To acccess the Background Graphic to Scale button, you will navigate to the Insert ribbon which contains tools necessary to create background graphics of various shapes, including the Background Graphic to Scale Button.

The Background Graphics to Scale allows you can select a background image, including DWG (requires AutoCAD® to be installed on your machine).

  • From the Insert Ribbon, you can select Background Graphic to Scale. The image can be zoomed in and out, and panned using your mouse and mouse wheel to find the correct section of the image that you want to place markers on to indicate distance and grid units desired. Click the starting point in the image and drag the mouse to the ending point and click. You can then select the distance and grid units you desire to be in that selection. The image will be placed in the model as the background scaled accordingly to your specifications.
  • The Model will then be scaled according to the selections you have made within the image when you click OK.
