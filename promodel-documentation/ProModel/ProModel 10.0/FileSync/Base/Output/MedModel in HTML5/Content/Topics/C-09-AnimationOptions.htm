View Controls

Controls for viewing different sections of your model during simulation as well as dynamically displaying statistics are found in the View group of the ribbon.

Zoom to Fit Select this option to bring all graphical elements of the model into view.

Zoom Select this option to Zoom in or out on the animation. When this option is selected, you may choose a preset zoom level or enter your own zoom level. The Zoom function zooms to the center of the screen.

To zoom in a specific area of the simulation layout, press and hold the CTRL key, then click and drag the mouse to create a rectangle around the are in which you would like to zoom. Release the mouse, then the CTRL key. The layout will zoom in on the selected area.

Views Panel Opens a window panel listing all defined views. Selecting a view from the list will zoom and pan the layout to show that view.

Views List Click on this menu item to display a list of the available views. Select the desired view to show it in the Layout.

Create Plot Allows you to create a plot of available statistics that dynamically updates as the simulation progresses.

Plot Configurations Opens a dialog for defining and saving configurations of one or more dynamic plots. These configurations can then be invoked in logic or manually opened through this same dialog.
