Network Client Installation

  1. Start Windows
  2. Insert CD-ROM or download the setup file from the Solutions Cafe to your desktop.
  3. While pressing the SHIFT key, right-click on the installation exe and select Copy as path from the context menu. This will copy the file name and path to your Windows clipboard.
  4. Open a Command Prompt window by pressing the Windows button + R, typing "cmd" in the Open field, and then selecting the OK button.
  5. Inside the Command Prompt window, right-click and select Paste. The file name and path you copied in step 3 should be shown.
  6. Enter a space after the file name and type the following: RUNTIMETYPE=Network.
  7. Press Enter and the installation will begin.
  8. If any prerequisite components are needed, the Setup will install them for you. There may be a need to restart the machine. If this is the case, restart the installation by going back to step 3.

  1. Click Next to proceed with the installation.

  1. Select Finish. Upon running MedModel, it will check your network for an available license.
