IncLocCost

Cost Related Statement

Syntax samples

IncLocCost <expression>

IncLocCost 15

IncLocCost -15

Description

Enables you to increment the cost (positively or negatively) of the current location by a given amount. Use this function to add to the location’s actual, accumulated dollar amount.

Valid In

Operation logic.

Components

<expression>

The positive or negative change to the value of cost.

Example

The logic below shows how to add a cost per entry to a location for an entering entity. MedModel automatically tracks operation cost per time unit specified in the cost dialog, however, you may assign a one time cost each time an entity enters a location.

In the following example, an entity, Patient, arrives at the location Triage. Rather than calculate an operational cost over time for Triage, we will apply a flat, one-time cost each time a nurse sees a patient—regardless of the time spent there. At Triage, a nurse examines the patient. At the end of the examination, MedModel adds a $12.00 cost to the location (for examining the patient) using the IncLocCost statement, then assigns that same cost to the Patient using the IncEntCost statement.
