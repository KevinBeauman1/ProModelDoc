Simulation Browser

The ProModel AutoCAD® Edition's Simulation Browser can be accessed from the ProModel Ribbon within AutoCAD®.

Once open, you will see the below tabs: Locations, Conveyors, Entities, Flows, Resources, and Path Networks.

Within each tab, you can add, delete, and reorder. Locations, Conveyors, Flow, Resources, and Path Networks you can also navigate to the location in the DWG.

Locations: The Locations Tab allows the user to add assets within the DWG by clicking the Add button and then clicking on the assets within the DWG then hitting Enter to commit the assets marked as locations in the model.

Once locations are added you can modify the name of the locations within the Browser or The Simulation Properties window as well as changing the capacity of each location, and add hourly cost to each location as well.

Conveyors: The Conveyors Tab allows the user to assign conveyor assets within the DWG to a conveyor that can be used in a Flow by clicking the Add button and following the command prompts:

  1. Enter the conveyor name.
  2. Select the assets that make up the conveyor.
  3. Choose the starting segment of the conveyor to designate the direction.

Once the conveyor is completed it will be displayed by name in the Simulation Browser. When selected from the Simulation Browser the conveyor, and all assets that make the conveyor will be highlighted in the DWG.

Once the conveyor is selected in the Simulation Browser, you can modify the name of the conveyor within the Browser or The Simulation Properties window as well as changing the capacity, change Speed, set the conveyor to be accumulating or non-accumulating, and specify the way the entities orientate on the conveyor (Widthwise, or Lengthwise).

Hourly cost can be applied to the conveyor as well.

Entities: The Entities Tab allows the user to add entities to the simulation. Entities can be added by selecting the Add button. They are also added from creating a flow.

Once an entity is added, you can add cost, along with Entity Width and Length (based on the DWG format it will be in feet or meters). Additionally you can choose the graphic and the color or the graphic shape for your entity/s.

Flows: The Flows Tab will allow the user to add a process flow allowing entities to flow from location to location, Process at those locations, and assign travel time to the next destination. There are 3 key parts of the flow along with their own Simulation Properties, each of which you have access to modify from right-click menus:

  • Flow - Users can add multiple flows to the drawing. each flow require one or more entities to be selected in the Simulation Properties
  • Process - Flows contain Process records. Each Process record is where at a given location ,entities process for a specified time. The Simulation Properties will allow you to set or modify these properties for each process record. Specify process time and if you need a resource to assist processing. Resources can have a priority of 0 to 10 and interrupt, which allows the resource to be preempted.
  • Route - Route allows the user to control where and how long an entity takes to get to a Destination Location. The Simulation Properties for Routes will allow you to set or modify these properties for each route record. Specify move time and if you need a resource to assist processing. Resources can have a priority of 0 to 10 and interrupt, which allows the resource to be preempted. If the resource is assigned to a path network, the move time will be ignored and the resource speed and the distance to the next destination will be used to calculate the movement from location to location.


Resources: The Resource Tab will allow the user to add a resource that can be used in the process flow, allowing entities to be processed with the use of a resource, or be transfer to the next location. Resources can be static or dynamic when placed on a path network. to add a resource you simply click add from the browser and select the Asset/s or Block/s in the drawing and hit enter to commit them. Resources on a path network have speed along with acceleration, deceleration, and pick up times.

Path Networks: ___FILL IN THE BLANK___The Path Networks Tab will allow the user to add a path for which resource can travel on to work or travel with entities to transport entities to and from destinations.

Path Networks are created from selecting the Add plus button in the ribbon and flowing the command prompts: name the path network then click on the layout to create nodes and connectors and interfaces.

Path networks can be passing on non passing (see Path Networks). Path networks are made up of nodes, connectors, and interfaces:

  • Nodes can be renamed and given a limit for the number of resources that can be at a node.
  • Connectors list the source and destination nodes for each connector. Additionally you can choose the connector to have Two Way or One Way direction. When using a One Way directional connector you can right click and flip the direction of the flow of the path connector.
  • Interfaces allow the user to specify what node the resource will go to, to interact with a location. Interfaces can be added to the model by selecting "Interface" in the command prompt and then selecting one or all the nodes and locations that you choose the network to interface with. Hitting "Enter" will connect the locations to the closest node to it. This interface can always be changed manually

To Create a Flow:

  • Select the Add Process Flow button from the Ribbon or from the Simulation Browser. The user must define the Flow name in the command bar in AutoCAD® then hit enter.
  • Enter the Entity name in the command bar in AutoCAD® then hit enter.
  • Click the Asset in the DWG file you want the Flow to start your flow. this will create your first Flow Process record. Next click the Destination of the next Location. This will create the Route to the next Process record along with the next Process record
  • Continue the process to continue the building your Process Flow. you can Branch or route to Exit from the command bar. and then hit enter to commit your flow.

Once the Flows have been built you can open the model in ProModel. You can also modify the DWG, by moving Assets, modifying Process times, Route times, Location Names, Entity Names, and reopen the model in ProModel.

Once in ProModel you can add additional complexity to your model and also view layout changed.
