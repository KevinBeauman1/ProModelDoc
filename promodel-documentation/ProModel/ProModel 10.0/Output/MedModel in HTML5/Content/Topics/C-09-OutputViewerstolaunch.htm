Output viewer(s) to launch

Output Viewer New models created in MedModel will choose Output Viewer as the default output viewer.

Minitab MedModel links and integrates with Minitab to provide users with Six Sigma analysis. This integration allows for the tracking of Six Sigma metrics in a MedModel model and the automatic reporting of these metrics in Minitab after a scenario is run.

Select the Minitab checkbox to automatically launch Minitab at the end of the simulation.

Please Note: At least one checkbox, with the option to have both checkboxes under the "Output viewer(s) to launch" area, need to be checked at all times.
