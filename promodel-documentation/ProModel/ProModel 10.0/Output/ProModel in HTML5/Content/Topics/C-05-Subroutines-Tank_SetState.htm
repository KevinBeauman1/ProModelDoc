Tank_SetState

Syntax samples

TANK_SETSTATE (<Tank ID>, <State>)

TANK_SETSTATE (TankA, Tank_Idle)

Description

Sets the state of the tank (e.g., Tank_State[<Tank ID>]) to a new state and updates the statistics since the last change of state.

Use Tank_SetState to explicitly change the state of a tank. Only use Tank_SetState if the default state changes do not adequately meet modeling needs.

Components

<Tank ID>

The tank name or index number.

<State>

The new state for the tank.
