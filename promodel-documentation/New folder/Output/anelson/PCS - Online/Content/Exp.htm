Exponential

Function

Syntax samples

Exp(<expression>)

Real1 = Exp(Real2)

Description

Returns the exponential of an expression. This function is equivalent to ex.

Valid In

Any Logic. This function returns a real number.

Components

<expression>

Exp() returns the exponential of this expression.
