Syntax Guide

The logic for a model can be constructed with valid statements and expressions within a logic window. Logic windows possess a number of features that facilitate the proper writing of logic, and one such feature is the Syntax Guide. The Syntax Guide works in conjunction with the Intellisense feature, and it becomes active when you type the full keyword of a statement within a logic window.

Once you complete typing a recognized keyword of a logic statement, it displays a guide to the proper syntax of that logic statement. The guide displays the components of the statement (both required and optional) and the syntax that is required to properly complete the statement. For those logic statements with multiple syntactic forms, the Syntax Guide will indicate that multiple options are available for selection. You can click the up or down arrows to view each of the options, and then click the syntactic form that you want to use.
