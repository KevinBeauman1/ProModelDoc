Using Alternative Resources

An activity or routing can use one of several different resources to perform the task.

 

The model used for this example, as well as all of the modeling "how-to" examples, can be found in the ProModel Solutions Café (http://www.promodel.com/solutionscafe/howto/). A video demonstrating solution A can be found here.

A video demonstrating solution B can be found here.

Solution A

  1. Define the resources to be used by dragging resource shapes onto the layout.
  2. Open the properties for the activity or routing.
  3. Click on Logic.
  4. In the Logic, define a USE statement specifying the alternative resources (use the OR option) and the time of use. (Note: resource is automatically freed after the usage time. To keep the resource until a subsequent activity, use a GET statement instead of a USE statement.)

Solution B

  1. Define the resources to be used by dragging resource shapes onto the layout.
  2. Select Resource Groups under the Process Simulator menu.
  3. Define a group name and add both of the defined resources to this group.
  4. In the Activity properties where the resources are to be used, enter the time of use in the Time field and select the resource group name in the Resource section.
Example

Patients entering a surgical department prep room are prepped by a nurse or a surgical assistant, whichever is available first. Prep time takes N(5,1) minutes.

Solution A

The logic indicates that the Nurse will be captured as the first preference, but, if not available, the Surgical Assistant is used.
