Round

Function

Syntax samples

Round(<expression>)

Int Var1 = Round(3.5)

Description

Rounds the expression to the nearest whole number. Use this function to override Process Simulator’s default truncation.

Valid In

Any logic. This function returns an integer.

Components

<expression>

The expression to be rounded.
