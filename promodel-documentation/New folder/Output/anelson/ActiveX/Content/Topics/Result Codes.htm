Result Codes

Almost all of the methods in the ProModelData object may optionally be written to return a result code. Result codes help you know if the execution of the method was successful. For a list of result codes, see Locations Table (1): PMD_Table_Location on page 1.

Definitions

Record Type = Table Number

 

Record = Row

 

Record Index = Row Number

 

Field = Column

 

Field Index = Column Number
