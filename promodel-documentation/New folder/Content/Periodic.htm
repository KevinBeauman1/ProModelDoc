Periodic

  • Entity   Shows the entity connected to the arrival connector and specifies the type of entities that will arrive. If an entity is not connected to the arrival connection, then this field will be enabled and the arrival entity is manually selected.
  • Occurrences   Number of times that this arrival will be repeated during the simulation.
  • Interval   Time lapsed between arrival occurrences.
  • Quantity   Number of entities that arrive at each occurrence.
  • Start Time   The time of the first arrival. After the first arrival all subsequent arrivals will arrive according to the Interval time. If no start time is defined, the first arrival will occur according to the Interval value.

Please note: The fields described above can accept distributions and variables.
