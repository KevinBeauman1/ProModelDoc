Updating Process Simulator

Periodically, ProModel releases updates to the Process Simulator program. If you have an Internet connection configured when running Process Simulator, Process Simulator automatically detects available updates, notifies you of changes, and then prompts you with the option to install them. You may also manually check for an update within Process Simulator.

How to manually check for updates in Process Simulator

  1. Open Process Simulator.
  2. From the Process Simulator ribbon, select Help > About and then the Updates button in the About window.

  1. If you are running the latest version of Process Simulator, a dialog box will appear notifying you.

  1. If not, the Update Available for Process Simulator dialog appears.
  2. Click the Next button (you may be asked to adjust your reminder settings). The Login Information dialog appears.
  3. In the Username and Password fields, type your ProModel username and password.
  4. If you do not have a valid ProModel username and/or password, click the hyperlink, “Forgot password?” to be redirected to the ProModel Solutions Cafe where you can request to have your username and password emailed to you.
  5. Click the Next button. The update utility will detect if a new version is available. If an update is available, continue following the on-screen instructions to complete the update.
