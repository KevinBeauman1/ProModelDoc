Macros

A macro is an element which can represent a distribution or a number that is repetitively used throughout your model. Assign the distribution or number to the macro and then enter the macro in most shape properties or logic, for example. Wherever the macro is encountered, the distribution or number it represents will be used instead. The macro can be used in many places in your model and when you want to change it, you only need to go to one place, the Macros grid. Macros can also be used as parameters in the Scenario Manager for scenario analysis.

Click the Macros button in the Model Elements group on the Process Simulator ribbon to set up Macros for your project.

  • Name The name of the macro.

Macro names must be alpha-numeric, or may contain underscores. No other special characters are allowed such as %, $, !, etc.

  • Type The type of expression the macro will represent, a distribution or number.
  • Value The distribution or numeric value of the macro. If the type is set as a distribution, double-click in the field to get into edit mode where you can change the distribution manually. If the type is set as a number, select the Value field and enter the number you want the macro to represent.
