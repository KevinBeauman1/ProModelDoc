Tracking Resource Activity Time Cost

Increase the unit cost of an entity according to the length of time a resource was used or activity performed.

 

The model used for this example, as well as all of the modeling "how-to" examples, can be found in the ProModel Solutions Café (http://www.promodel.com/solutionscafe/howto/).

Solution

  1. Open the resource or activity dialog.
  2. Enter an hourly cost to be applied.
  3. Each time during the simulation when this resource or activity is used, the hourly rate is used to determine the incremental cost to be added to the entity.
Example

A coordinate measuring machine (CMM) is used on certain parts and is charged out at $150.00 an hour. We want to add this cost to the entities that use it.

Solution

The hourly rate for the CMM is set to $150. Alternatively, we could have defined the CMM as an activity and set the hourly rate for the activity to $150.
