Trunc

Function

Syntax samples

Trunc(<expression>)

Int Var1 = Trunc(3.9)

Description

Returns a real expression truncated to an integer. Any digits to the right of the decimal place will be removed. When necessary, Process Simulator automatically converts real values to integers by truncating them.

Valid In

Any logic. This function returns an integer.

Components

<expression>

The expression to be truncated.
