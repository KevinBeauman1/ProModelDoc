Using the Help System

The Process Simulator Help System is designed to be a reference to guide you through the process of building simulation models in Visio. While the Help System can be used as a step-by-step guide to model building by following the table of contents order, it is much more functional as a reference while you are building a model.

Navigating the Help System

The Help System is most easily navigated using the table of contents along the left side of the help window. Sections are divided into topics. Additionally, the Help System can be navigated using the right and left arrows in the table of contents to browse forward and backward through the table of contents. The Back button located in the Help toolbar takes you to the previous page you viewed, which may not always be the previous page in the table of contents.

Advanced use of the Help System

The quickest way to use the Help System is to select Help from the Process Simulator ribbon, which will launch the online Help System. You can then browse the Table of Contents or enter a word or phrase in the Search field in order to find an answer to any Process Simulator questions you may have.

If you are unable to find an answer to your question in this Help System, please feel free to contact Technical Support.
