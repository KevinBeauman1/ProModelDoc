Flexible

  • Destination   Enter an attribute, array, or variable that will resolve into an existing activity name (i.e. an activity index number) or right-click to view the context menu showing the available data elements. This determines the next activity to which an entity will route, and the destination is independently evaluated for each entity taking the route. Clicking the Destination drop down list will show a list of activities from which you can choose to route to.

Please note: To use flexible routing, you must not connect the head of the routing arrow to any specific activity; however, the arrow head can be connected to a plain graphic in the layout.
