Ordered

An arrival with an Ordered type will not arrive until an order is triggered by the conditions defined in a Storage activity and the Lead Time is exhausted for the entity. For more information on ordered arrivals in connection with Storage activities, see Storage Properties.

Entity  Shows the entity connected to the arrival connector and specifies the type of entities that will arrive. If an entity is not connected to the arrival connection, then this field will be enabled and the arrival entity is manually selected. 

Lead Time   Enter the time between the triggering of the order and the arrival of the entities at the activity.
