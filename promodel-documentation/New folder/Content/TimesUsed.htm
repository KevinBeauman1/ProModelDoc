TimesUsed

Function

Syntax samples

TimesUsed(<resource>)

If TimesUsed(Res1) > 5 Then Use Res2 for 10

Description

Returns the number of times a resource has been used.

Valid In

Any logic.

Components

<resource>

The name of the resource to examine. Res() may be used here.

Example

In the example below, customers arrive at a local copy shop. Two resources (Trainee and Mgr) are available to service customers. If the trainee has serviced 10 or fewer clients, the manager must assist in helping the customer. Otherwise, the Trainee may help customers without the assistance of Mgr.

If TimesUsed(Trainee) <= 10 Then
{

Get Manager And Trainee
Wait 10 min
Free All

}

Else

Use Trainee For 10 min
