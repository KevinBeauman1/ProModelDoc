Entries()

Function

Syntax samples

Entries(<activity>)

Display “Act1 has had” $ Entries(Act1) $ “entries.”

Description

Returns the total number of entity entries to an activity. This function returns an integer.

Valid In

Any logic

Components

<activity>

The activity to examine.

Example

The Stores activity sends Order entities to Shipping. A variable, Batch_Time, tracks the amount of time it takes to process 50 Orders. After Assembly processes 50 entities, we want to reset Batch_Time to zero.

If Entries(Assembly) = 50 Then

Batch_Time = 0
